## 🎮 情景引入

> 想象你在玩一个 RPG 游戏，角色有一个**经验值系统**：
> 
> - 每打一只怪，获得固定经验值
> - 等级 = 经验值 ÷ 100（取整）
> 
> 这里有一个**对应关系**：
> ```
> 经验值 → 等级
>   0    →   1
>  100   →   2
>  200   →   3
>  520   →   6
> ```
> 
> **关键特性**：
> - ✅ 每个经验值**对应唯一**的等级
> - ❌ 但一个等级可能对应多个经验值（100~199 都是 2 级）
> 
> 这就是**函数**！在信竞中，函数无处不在——从数学公式到算法复杂度，从状态转移到 DP 方程。

## 📖 直观理解

### 1. 什么是函数？

**函数**是一种特殊的对应关系：

```
定义域（输入） → 对应法则 → 值域（输出）
     x        →    f     →    f(x)
```

**关键特性**：
- ✅ **唯一性**：每个输入对应**唯一**的输出
- ✅ **确定性**：相同的输入一定得到相同的输出

**示例**：

| 是函数 | 不是函数 |
|--------|----------|
| 学号 → 姓名 | 姓名 → 学号（重名） |
| 时间 → 温度 | 温度 → 时间（多时刻） |
| x → x² | x → ±√x（不唯一） |

### 2. 函数的三要素

{% mermaid %}
graph TD
    A[函数] --> B1[定义域]
    A --> B2[对应法则]
    A --> B3[值域]
    
    B1 --> C1["x 的取值范围"]
    B2 --> C2["f: x → f(x)"]
    B3 --> C3["f(x) 的取值范围"]
{% endmermaid %}

**1. 定义域**：自变量 x 的取值范围

```
f(x) = 1/x  的定义域是 {x | x ≠ 0}
f(x) = √x   的定义域是 {x | x ≥ 0}
```

**2. 对应法则**：输入到输出的映射规则

```
f(x) = x² + 1
g(x) = 2x - 3
h(x) = sin(x)
```

**3. 值域**：函数值 f(x) 的取值范围

```
f(x) = x²  的值域是 {y | y ≥ 0}
f(x) = 1/x 的值域是 {y | y ≠ 0}
```

### 3. 函数的表示方法

**1. 解析法**：用公式表示

```
f(x) = x² + 2x + 1
```

**2. 列表法**：用表格表示

| x | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| f(x) | 2 | 4 | 6 | 8 | 10 |

**3. 图像法**：用图像表示

```
    y
    |     /
    |    /
    |   /
    |  /
    | /
    +-------- x
```

**4. 信竞中的表示**：函数作为代码

```cpp
// C++ 中的函数
int f(int x) {
    return x * x + 2 * x + 1;
}
```

## 📐 数学定义

### 函数的正式定义

设 A、B 是两个非空数集，如果按照某种**确定的对应关系 f**，使对于集合 A 中的**任意一个数 x**，在集合 B 中都有**唯一确定的数 f(x)** 和它对应，那么就称 f: A → B 为从集合 A 到集合 B 的一个函数。

记作：
$$
y = f(x), \quad x \in A
$$

其中：
- x 叫做**自变量**
- y 叫做**因变量**（或函数值）
- A 叫做函数的**定义域**
- {f(x) | x ∈ A} 叫做函数的**值域**

### 常见函数类型

| 函数类型 | 解析式 | 定义域 | 图像特征 |
|---------|--------|--------|---------|
| 一次函数 | f(x) = kx + b | R | 直线 |
| 二次函数 | f(x) = ax² + bx + c | R | 抛物线 |
| 反比例函数 | f(x) = k/x | x≠0 | 双曲线 |
| 幂函数 | f(x) = x^a | 视 a 而定 | 多样 |
| 绝对值函数 | f(x) = \|x\| | R | V 字形 |

### 分段函数

在定义域的不同部分，有不同的对应法则：

$$
f(x) = \begin{cases} 
x, & x < 0 \\
x^2, & 0 \leq x < 1 \\
2x - 1, & x \geq 1
\end{cases}
$$

**信竞中的应用**：

```cpp
int piecewise(int x) {
    if (x < 0) {
        return x;
    } else if (x < 1) {
        return x * x;
    } else {
        return 2 * x - 1;
    }
}
```

### 复合函数

**定义**：一个函数的输出作为另一个函数的输入

$$
(f \circ g)(x) = f(g(x))
$$

**示例**：
```
f(x) = x²
g(x) = x + 1

则：f(g(x)) = (x + 1)²
    g(f(x)) = x² + 1
```

**注意**：复合函数**不满足交换律**！

## 💻 代码实现

### 函数的 C++ 实现

```cpp
#include <bits/stdc++.h>
using namespace std;

// 1. 基本函数定义
int square(int x) {
    return x * x;
}

// 2. 带默认参数的函数
int power(int x, int n = 2) {
    int result = 1;
    for (int i = 0; i < n; i++) {
        result *= x;
    }
    return result;
}

// 3. 分段函数
int piecewise(int x) {
    if (x < 0) return x;
    else if (x < 1) return x * x;
    else return 2 * x - 1;
}

// 4. 递归函数（斐波那契）
int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

// 5. 函数作为参数（回调函数）
int apply(int x, int (*func)(int)) {
    return func(x);
}

int main() {
    cout << square(5) << endl;        // 25
    cout << power(3) << endl;         // 9 (默认平方)
    cout << power(2, 3) << endl;      // 8
    cout << piecewise(-2) << endl;    // -2
    cout << fibonacci(10) << endl;    // 55
    cout << apply(4, square) << endl; // 16
    
    return 0;
}
```

### 函数在信竞中的应用

**1. 数学公式直接实现**

```cpp
// 计算两点间距离
double distance(double x1, double y1, double x2, double y2) {
    return sqrt((x2 - x1) * (x2 - x1) + (y2 - y1) * (y2 - y1));
}

// 计算组合数 C(n, m)
long long combination(int n, int m) {
    if (m > n) return 0;
    if (m == 0 || m == n) return 1;
    
    long long result = 1;
    for (int i = 1; i <= m; i++) {
        result = result * (n - i + 1) / i;
    }
    return result;
}
```

**2. 状态转移函数（DP）**

```cpp
// 最长上升子序列的状态转移
int lis(const vector<int>& nums) {
    int n = nums.size();
    vector<int> dp(n, 1);  // dp[i] 表示以 nums[i] 结尾的 LIS 长度
    
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
    }
    
    return *max_element(dp.begin(), dp.end());
}
```

**3. 哈希函数**

```cpp
// 简单的字符串哈希函数
unsigned long long hash_string(const string& s) {
    unsigned long long hash = 0;
    unsigned long long base = 131;
    
    for (char c : s) {
        hash = hash * base + c;
    }
    
    return hash;
}

// 判断两个字符串是否相等（哈希比较）
bool equal_hash(const string& a, const string& b) {
    return hash_string(a) == hash_string(b);
}
```

### 函数的时间复杂度分析

```cpp
// O(1) - 常数时间
int constant(int x) {
    return x * x;
}

// O(n) - 线性时间
int linear(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
}

// O(n²) - 平方时间
int quadratic(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            sum += i * j;
        }
    }
    return sum;
}

// O(log n) - 对数时间
int logarithmic(int n) {
    int count = 0;
    while (n > 1) {
        n /= 2;
        count++;
    }
    return count;
}

// O(2^n) - 指数时间
int exponential(int n) {
    if (n <= 1) return 1;
    return exponential(n - 1) + exponential(n - 1);
}
```

## 🏆 信竞应用

### 1. 函数思想在算法中的应用

**二分查找**：利用函数的单调性

```cpp
// 在有序数组中查找目标值
int binary_search(const vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] == target) {
            return mid;  // 找到
        } else if (nums[mid] < target) {
            left = mid + 1;  // 在右半部分
        } else {
            right = mid - 1;  // 在左半部分
        }
    }
    
    return -1;  // 未找到
}
```

**核心思想**：将查找问题转化为函数值比较问题。

### 2. 递推关系（递归函数）

**斐波那契数列**：

$$
F(n) = \begin{cases}
0, & n = 0 \\
1, & n = 1 \\
F(n-1) + F(n-2), & n \geq 2
\end{cases}
$$

```cpp
// 递归实现（指数时间）
int fib_recursive(int n) {
    if (n <= 1) return n;
    return fib_recursive(n - 1) + fib_recursive(n - 2);
}

// 迭代实现（线性时间）
int fib_iterative(int n) {
    if (n <= 1) return n;
    
    int a = 0, b = 1;
    for (int i = 2; i <= n; i++) {
        int c = a + b;
        a = b;
        b = c;
    }
    return b;
}

// 矩阵快速幂（对数时间）
vector<vector<long long>> multiply(const vector<vector<long long>>& a,
                                   const vector<vector<long long>>& b) {
    vector<vector<long long>> c(2, vector<long long>(2));
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            for (int k = 0; k < 2; k++) {
                c[i][j] += a[i][k] * b[k][j];
            }
        }
    }
    return c;
}

int fib_matrix(int n) {
    if (n <= 1) return n;
    
    vector<vector<long long>> result = {{1, 0}, {0, 1}};  // 单位矩阵
    vector<vector<long long>> base = {{1, 1}, {1, 0}};
    
    while (n > 0) {
        if (n & 1) result = multiply(result, base);
        base = multiply(base, base);
        n >>= 1;
    }
    
    return result[0][1];
}
```

### 3. 函数在数据结构中的应用

**栈的应用**：函数调用栈

```cpp
// 模拟函数调用栈
stack<pair<string, int>> call_stack;

void call_function(string name, int param) {
    call_stack.push({name, param});
    cout << "调用 " << name << "(" << param << ")" << endl;
}

void return_function() {
    if (!call_stack.empty()) {
        auto [name, param] = call_stack.top();
        call_stack.pop();
        cout << "返回 " << name << "(" << param << ")" << endl;
    }
}
```

### 4. 映射（Map）作为函数

```cpp
// 用 map 表示离散函数
map<int, int> f;

// 定义函数值
f[1] = 2;
f[2] = 4;
f[3] = 6;

// 查询函数值
if (f.count(2)) {

// 遍历函数
for (auto [x, y] : f) {
    cout << "f(" << x << ") = " << y << endl;
}
```

## 📚 拓展阅读

### 进阶知识（选学）

1. **一一映射（双射）**
   - 既是单射（不同输入对应不同输出）
   - 又是满射（值域等于到达域）
   - 双射函数有反函数

2. **反函数**
   - 如果 f: A → B 是双射，则存在反函数 f⁻¹: B → A
   - f(f⁻¹(x)) = x, f⁻¹(f(x)) = x
   - 图像关于 y = x 对称

3. **函数的连续性**
   - 直观理解：图像可以一笔画成
   - 严格定义：lim(x→a) f(x) = f(a)

4. **函数的极限**
   - 描述函数在某点附近的变化趋势
   - 微积分的基础
