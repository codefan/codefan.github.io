## 🎯 例题精讲

### 例题 1：集合的交集与并集

**题目链接**: [洛谷 P2357 - 集合并集](https://www.luogu.com.cn/problem/P2357)

**题意分析**: 

给定两个集合 A 和 B，每个集合包含 n 个整数。请计算：
1. A ∩ B（交集）
2. A ∪ B（并集）

**输入格式**:
```
第一行：n (1 ≤ n ≤ 10^5)
第二行：n 个整数，表示集合 A
第三行：n 个整数，表示集合 B
```

**输出格式**:
```
第一行：交集中元素的个数
第二行：并集中元素的个数
```

**解题思路**: 

1. 使用 C++ STL 的 `set` 自动去重
2. 用 `set_intersection` 和 `set_union` 计算交集和并集
3. 注意：结果也需要去重

{% mermaid %}
graph LR
    A[读入数据] --> B[存入 set 去重]
    B --> C[计算交集]
    B --> D[计算并集]
    C --> E[输出结果]
    D --> E
{% endmermaid %}

**参考代码**:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    int n;
    cin >> n;
    
    set<int> A, B;
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        A.insert(x);
    }
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        B.insert(x);
    }
    
    // 计算交集
    vector<int> intersection;
    set_intersection(A.begin(), A.end(),
                     B.begin(), B.end(),
                     back_inserter(intersection));
    
    // 计算并集
    vector<int> union_set;
    set_union(A.begin(), A.end(),
              B.begin(), B.end(),
              back_inserter(union_set));
    
    cout << intersection.size() << "\n";
    cout << union_set.size() << "\n";
    
    return 0;
}
```

**复杂度分析**:
- 时间复杂度：O(n log n)（set 插入和遍历）
- 空间复杂度：O(n)

### 例题 2：逻辑命题的真假判断

**题目链接**: [AtCoder ABC 285 A - 逻辑判断练习](https://atcoder.jp/contests/abc285/tasks/abc285_a)

**题意分析**: 

给定三个布尔值 A, B, C，判断以下命题的真假：
1. A ∧ B ∧ C（全为真）
2. A ∨ B ∨ C（至少一个为真）
3. ¬A ∨ ¬B（A 和 B 不同时为真）

**输入格式**:
```
一行三个整数 (0 或 1)，表示 A, B, C
```

**输出格式**:
```
三行，每行输出 Yes 或 No
```

**解题思路**: 

直接使用逻辑运算符：
- `&&` 表示 ∧（且）
- `||` 表示 ∨（或）
- `!` 表示 ¬（非）

**参考代码**:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;
    
    // 命题 1: A ∧ B ∧ C
    if (A && B && C) {
        cout << "Yes\n";
    } else {
        cout << "No\n";
    }
    
    // 命题 2: A ∨ B ∨ C
    if (A || B || C) {
        cout << "Yes\n";
    } else {
        cout << "No\n";
    }
    
    // 命题 3: ¬A ∨ ¬B
    if ((!A) || (!B)) {
        cout << "Yes\n";
    } else {
        cout << "No\n";
    }
    
    return 0;
}
```

### 例题 3：状态压缩入门

**题目链接**: [洛谷 P1433 - 吃奶酪](https://www.luogu.com.cn/problem/P1433)（简化版）

**题意分析**: 

有 n 块奶酪，老鼠从原点出发，要吃掉所有奶酪。问最短路径是多少？

**数据范围**: n ≤ 15（提示：状态压缩）

**解题思路**: 

1. 用二进制位表示"已吃过的奶酪集合"
2. `dp[mask][i]` = 吃过 mask 表示的奶酪集合，最后在奶酪 i 的最短路径
3. 状态转移：枚举下一个要吃的奶酪 j

**参考代码**:

```cpp
#include <bits/stdc++.h>
using namespace std;

const double INF = 1e18;

int n;
double x[20], y[20];
double dp[1 << 15][15];

double dist(int i, int j) {
    return sqrt((x[i] - x[j]) * (x[i] - x[j]) + 
                (y[i] - y[j]) * (y[i] - y[j]));
}

int main() {
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> x[i] >> y[i];
    }
    
    // 初始化
    for (int mask = 0; mask < (1 << n); mask++) {
        for (int i = 0; i < n; i++) {
            dp[mask][i] = INF;
        }
    }
    
    // 从原点出发到每个奶酪
    for (int i = 0; i < n; i++) {
        dp[1 << i][i] = dist(-1, i);  // 从原点到奶酪 i
    }
    
    // 状态转移
    for (int mask = 1; mask < (1 << n); mask++) {
        for (int i = 0; i < n; i++) {
            if ((mask >> i) & 1) {  // 奶酪 i 在集合中
                if (dp[mask][i] == INF) continue;
                
                for (int j = 0; j < n; j++) {
                    if (!((mask >> j) & 1)) {  // 奶酪 j 不在集合中
                        int new_mask = mask | (1 << j);
                        dp[new_mask][j] = min(dp[new_mask][j],
                                              dp[mask][i] + dist(i, j));
                    }
                }
            }
        }
    }
    
    // 答案：吃完所有奶酪的最短路径
    double ans = INF;
    int full_mask = (1 << n) - 1;
    for (int i = 0; i < n; i++) {
        ans = min(ans, dp[full_mask][i]);
    }
    
    cout << fixed << setprecision(2) << ans << "\n";
    
    return 0;
}
```

**关键点**:
- `mask` 的二进制第 i 位为 1 表示"已吃奶酪 i"
- 状态数：2^n × n，n=15 时约 50 万，可以接受

### 习题 1：集合的基本运算

**题目**: 给定 A = {1,2,3,4,5}, B = {4,5,6,7,8}，求 A∩B 和 A∪B

**答案**: 
- A ∩ B = {4, 5}
- A ∪ B = {1, 2, 3, 4, 5, 6, 7, 8}

### 习题 2：命题真假判断

**题目**: 判断下列命题的真假
1. "若 x > 2，则 x² > 4"
2. "存在实数 x，使得 x² + 1 = 0"
3. "对所有整数 n，n² 是偶数当且仅当 n 是偶数"

**答案**: 
1. ✅ 真（x > 2 时，x² > 4 恒成立）
2. ❌ 假（实数范围内 x² ≥ 0，x² + 1 ≥ 1 ≠ 0）
3. ✅ 真（偶数的平方是偶数，奇数的平方是奇数）

## 📊 本章小结

### 重点回顾

1. **集合三特性**：确定性、互异性、无序性
2. **四种运算**：
   - 交集：A ∩ B（共有元素）
   - 并集：A ∪ B（所有元素，去重）
   - 补集：∁ᵤA（不属于 A 的元素）
   - 差集：A - B（在 A 中但不在 B 中）
3. **逻辑联结词**：∧（且）、∨（或）、¬（非）
4. **量词**：∀（全称）、∃（存在）
5. **位运算表示集合**：状态压缩的核心思想

### 易错点提醒

⚠️ **集合并集要去重**：{1,2,3} ∪ {2,3,4} = {1,2,3,4}（不是 {1,2,3,2,3,4}）

⚠️ **空集是任何集合的子集**：∅ ⊆ A 恒成立

⚠️ **量词否定要变号**：
- "所有人都会" 的否定是 "有人不会"（不是 "所有人都不会"）
- "有人会" 的否定是 "所有人都不会"
