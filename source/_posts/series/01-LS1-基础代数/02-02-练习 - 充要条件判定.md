## 🎯 例题精讲

### 例题 1：充要条件的判断

**题目来源**: 课本习题

**题意分析**: 

判断下列各题中 p 是 q 的什么条件：

1. p: x > 2, q: x > 0
2. p: x² > 4, q: x > 2
3. p: 三角形是等边三角形，q: 三角形是等角三角形
4. p: a 是偶数，q: a² 是偶数

**解题思路**: 

对每个小题：
1. 判断 p → q 是否为真
2. 判断 q → p 是否为真
3. 根据真值表得出结论

**参考答案**:

1. **p: x > 2, q: x > 0**
   - p → q: 若 x > 2，则 x > 0 ✅（真）
   - q → p: 若 x > 0，则 x > 2 ❌（假，如 x=1）
   - **结论**: 充分不必要条件

2. **p: x² > 4, q: x > 2**
   - p → q: 若 x² > 4，则 x > 2 ❌（假，如 x=-3）
   - q → p: 若 x > 2，则 x² > 4 ✅（真）
   - **结论**: 必要不充分条件

3. **p: 等边三角形，q: 等角三角形**
   - p → q: 等边 → 等角 ✅（真）
   - q → p: 等角 → 等边 ✅（真）
   - **结论**: 充要条件

4. **p: a 是偶数，q: a² 是偶数**
   - p → q: 偶数 → 平方是偶数 ✅（真）
   - q → p: 平方是偶数 → 原数是偶数 ✅（真，奇数的平方是奇数）
   - **结论**: 充要条件

### 例题 2：集合观点看条件

**题目来源**: 奥数题

**题意分析**: 

已知集合 A = {x | x > 2}, B = {x | x > 0}

问："x ∈ A" 是 "x ∈ B" 的什么条件？

**解题思路**: 

用集合观点分析：
1. 画出集合 A 和 B 的数轴表示
2. 判断集合包含关系
3. 得出结论

**参考答案**:

```
数轴表示：
B: |--------●================>
             0

A:            |--------●=====>
                      2

显然 A ⊂ B（A 是 B 的真子集）

根据集合观点：
- A ⊆ B → "x∈A" 是 "x∈B" 的充分条件
- B ⊄ A → "x∈A" 不是 "x∈B" 的必要条件

结论：充分不必要条件
```

### 例题 3：二分答案中的条件判断

**题目来源**: 洛谷

**题意分析**: 

给定 n 个数，求第 k 小的数。

使用二分答案时，需要找到"答案 ≤ mid"的充要条件。

**输入格式**:
```
第一行：n, k
第二行：n 个整数
```

**输出格式**:
```
一个整数，表示第 k 小的数
```

**解题思路**: 

关键：找到"第 k 小的数 ≤ mid"的充要条件

充要条件：**≤ mid 的数的个数 ≥ k**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    // 二分答案
    int left = -1e9, right = 1e9;
    
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        // 充要条件：≤ mid 的数的个数 ≥ k
        int count = 0;
        for (int x : a) {
            if (x <= mid) count++;
        }
        
        if (count >= k) {
            right = mid;
        } else {
            left = mid + 1;
        }
    }
    
    cout << left << endl;
    
    return 0;
}
```

### 例题 4：图论中的充要条件应用

**题目来源**: 洛谷

**题意分析**: 

判断一个无向图是否存在欧拉回路。

**输入格式**:
```
第一行：n, m（顶点数，边数）
接下来 m 行：每行两个整数 u, v，表示一条边
```

**输出格式**:
```
如果存在欧拉回路，输出 "YES"，否则输出 "NO"
```

**解题思路**: 

欧拉回路存在的**充要条件**：
1. 所有顶点的度数都是偶数
2. 图是连通的（忽略孤立点）

**参考思路**:
```cpp
// 提示：
// 1. 统计每个顶点的度数
// 2. 检查是否全为偶数
// 3. 用 DFS/BFS 检查连通性
// 4. 两个条件都满足才有欧拉回路
```

## 📊 本章小结

### 重点回顾

1. **三种条件的定义**：
   - 充分条件：p → q
   - 必要条件：q → p
   - 充要条件：p ↔ q

2. **判定方法**：
   - 定义法：直接判断两个方向
   - 集合法：看集合包含关系
   - 等价法：利用逆否命题

3. **四种分类**：
   - 充要条件（✅✅）
   - 充分不必要（✅❌）
   - 必要不充分（❌✅）
   - 既不充分也不必要（❌❌）

4. **信竞应用**：
   - 二分答案的充要条件
   - 图论判定（欧拉回路、二分图）
   - 贪心算法的正确性证明
   - 搜索剪枝的必要条件

### 易错点提醒

⚠️ **充分 vs 必要**：
- "如果 p，那么 q" → p 是 q 的充分条件
- "只有 p，才 q" → p 是 q 的必要条件

⚠️ **集合关系方向**：
- A ⊆ B → "x∈A" 是 "x∈B" 的充分条件
- 方向容易搞反！

⚠️ **逆否命题等价**：
- p → q 等价于 ¬q → ¬p
- 直接证明困难时，可以证明逆否命题

⚠️ **必要条件用于剪枝**：
- 不满足必要条件 → 一定无解 → 可以剪枝
- 满足必要条件 → 不一定有解 → 继续搜索
