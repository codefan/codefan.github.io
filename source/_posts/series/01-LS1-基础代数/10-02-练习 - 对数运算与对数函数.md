## 🎯 例题精讲

### 例题 1：对数运算

**题目来源**: 课本习题

**题意分析**: 

计算：
1. log₂(8)
2. log₃(27)
3. log₁₀(1000)
4. log₂(1/4)

**参考答案**:

1. log₂(8) = log₂(2³) = 3
2. log₃(27) = log₃(3³) = 3
3. log₁₀(1000) = log₁₀(10³) = 3
4. log₂(1/4) = log₂(2⁻²) = -2

---

### 例题 2：对数方程

**题目来源**: 自编

**题意分析**: 

解方程：log₂(x) + log₂(x - 2) = 3

**解题思路**: 

利用对数运算法则化简。

**参考答案**:

```
log₂(x) + log₂(x - 2) = 3
log₂(x(x - 2)) = 3
x(x - 2) = 2³ = 8
x² - 2x - 8 = 0
(x - 4)(x + 2) = 0

解得：x = 4 或 x = -2

检验：x > 0 且 x - 2 > 0，所以 x = 4

答案：x = 4
```

## 📊 本章小结

### 重点回顾

1. **对数定义**：a^x = N ⟺ x = logₐ(N)
2. **运算法则**：乘法变加法，除法变减法
3. **换底公式**：logₐ(b) = logₖ(b) / logₖ(a)
4. **对数函数**：定义域 (0, +∞)，过定点 (1, 0)

### 易错点提醒

⚠️ **定义域**：logₐ(x) 要求 x > 0

⚠️ **底数限制**：a > 0 且 a ≠ 1

⚠️ **运算顺序**：log(M + N) ≠ log(M) + log(N)

### 信竞应用总结

| 知识点 | 应用场景 | 典型题目 |
|--------|---------|---------|
| 对数计算 | 数值计算 | 科学计算 |
| 换底公式 | 任意底数 | 计算器实现 |
| O(log n) | 复杂度分析 | 二分查找 |
| 阶乘位数 | 大数问题 | 组合数学 |
