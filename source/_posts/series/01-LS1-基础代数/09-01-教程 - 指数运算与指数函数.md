## 🎮 情景引入

> 想象你在玩一个**病毒传播模拟游戏**：
> 
> **场景**：一个病毒，每小时分裂一次
> ```
> 时间 (小时)    病毒数量
>     0           1
>     1           2
>     2           4
>     3           8
>     ...
>     t           2^t
> ```
> 
> **问题**：10 小时后有多少病毒？
> ```
> 2^10 = 1024 个
> ```
> 
> **反向问题**：病毒达到 100 万个需要多少小时？
> ```
> 2^t = 1000000
> t = log₂(1000000) ≈ 20 小时
> ```
> 
> **信竞中的应用**：指数增长在复杂度分析、数论、组合计数中无处不在！

## 📖 直观理解

### 1. 指数运算的推广

**整数指数**：
```
aⁿ = a × a × ... × a (n 个 a 相乘)
a⁰ = 1 (a ≠ 0)
a⁻ⁿ = 1/aⁿ (a ≠ 0)
```

**分数指数**：
```
a^(1/n) = ⁿ√a (n 次根号)
a^(m/n) = ⁿ√(aᵐ)
```

**实数指数**：
```
a^x (x 为任意实数，a > 0)
通过极限定义
```

### 2. 指数函数

**指数函数**：f(x) = a^x (a > 0, a ≠ 1)

**图像特征**：
```

    y               y
    |  /            |  \
    | /             |   \
    |/              |    \
____|____ x    _____|_____\___ x
    |0              |0
```

**共同性质**：
- 定义域：R
- 值域：(0, +∞)
- 过定点：(0, 1)
- 永远在 x 轴上方

## 📐 数学定义

### 指数运算法则

设 a > 0, b > 0, m, n ∈ R：

| 法则 | 公式 | 示例 |
|------|------|------|
| 乘法 | aᵐ × aⁿ = aᵐ⁺ⁿ | 2³ × 2⁴ = 2⁷ |
| 除法 | aᵐ ÷ aⁿ = aᵐ⁻ⁿ | 2⁵ ÷ 2² = 2³ |
| 幂的幂 | (aᵐ)ⁿ = aᵐⁿ | (2³)² = 2⁶ |
| 积的幂 | (ab)ⁿ = aⁿbⁿ | (2×3)² = 2²×3² |
| 商的幂 | (a/b)ⁿ = aⁿ/bⁿ | (4/2)³ = 4³/2³ |

### 指数函数的性质

| 性质 | a > 1 | 0 < a < 1 |
|------|-------|-----------|
| 单调性 | 单调递增 | 单调递减 |
| 值域 | (0, +∞) | (0, +∞) |
| 定点 | (0, 1) | (0, 1) |
| 渐近线 | y = 0 (x → -∞) | y = 0 (x → +∞) |
| 示例 | y = 2^x | y = (1/2)^x |

## 💻 代码实现

### 指数运算

```cpp
#include <bits/stdc++.h>
using namespace std;

// 整数指数（快速幂）
long long power_int(long long a, int n) {
    if (n < 0) return 0;
    if (n == 0) return 1;
    
    long long result = 1;
    while (n > 0) {
        if (n & 1) result *= a;
        a *= a;
        n >>= 1;
    }
    return result;
}

// 浮点指数
double power_double(double a, double x) {
    return pow(a, x);
}

// 指数增长模拟
vector<long long> exponential_growth(long long initial, double rate, int steps) {
    vector<long long> result;
    double current = initial;
    
    for (int i = 0; i <= steps; i++) {
        result.push_back((long long)current);
        current *= rate;
    }
    
    return result;
}

int main() {
    // 快速幂
    cout << power_int(2, 10) << endl;  // 1024
    
    // 浮点指数
    cout << power_double(2, 0.5) << endl;  // 1.414
    
    // 指数增长
    auto growth = exponential_growth(1, 2, 10);
    for (int i = 0; i <= 10; i++) {
        cout << "t=" << i << ": " << growth[i] << endl;
    }
    
    return 0;
}
```

### 模意义下的指数运算

```cpp
// 快速幂（模意义下）
long long power_mod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    
    while (exp > 0) {
        if (exp & 1) {
            result = result * base % mod;
        }
        base = base * base % mod;
        exp >>= 1;
    }
    
    return result;
}

// 费马小定理求逆元
// 如果 p 是质数，则 a^(p-2) ≡ a^(-1) (mod p)
long long mod_inverse(long long a, long long p) {
    return power_mod(a, p - 2, p);
}

int main() {
    long long mod = 1e9 + 7;
    
    // 计算 2^1000000 mod (10^9 + 7)
    cout << power_mod(2, 1000000, mod) << endl;
    
    // 计算 3 的逆元
    cout << mod_inverse(3, mod) << endl;
    
    return 0;
}
```

### 2. 快速幂应用

```cpp
// 计算斐波那契数列（矩阵快速幂）
vector<vector<long long>> multiply(const vector<vector<long long>>& a,
                                   const vector<vector<long long>>& b,
                                   long long mod) {
    vector<vector<long long>> c(2, vector<long long>(2));
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            for (int k = 0; k < 2; k++) {
                c[i][j] = (c[i][j] + a[i][k] * b[k][j]) % mod;
            }
        }
    }
    return c;
}

long long fibonacci_matrix(int n, long long mod) {
    if (n <= 1) return n;
    
    vector<vector<long long>> result = {{1, 0}, {0, 1}};
    vector<vector<long long>> base = {{1, 1}, {1, 0}};
    
    while (n > 0) {
        if (n & 1) result = multiply(result, base, mod);
        base = multiply(base, base, mod);
        n >>= 1;
    }
    
    return result[0][1];
}
```

### 3. 组合数计算

```cpp
// 组合数 C(n, m) mod p
long long combination_mod(int n, int m, long long p) {
    if (m > n) return 0;
    if (m == 0 || m == n) return 1;
    
    // 预计算阶乘
    vector<long long> fact(n + 1);
    fact[0] = 1;
    for (int i = 1; i <= n; i++) {
        fact[i] = fact[i - 1] * i % p;
    }
    
    // C(n, m) = n! / (m! × (n-m)!)
    long long numerator = fact[n];
    long long denominator = fact[m] * fact[n - m] % p;
    
    return numerator * mod_inverse(denominator, p) % p;
}
```
