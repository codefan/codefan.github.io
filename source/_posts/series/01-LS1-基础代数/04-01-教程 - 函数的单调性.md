## 🎮 情景引入

> 想象你在玩一个**爬塔游戏**，塔有 100 层：
> 
> **场景 1**：每层高度相同
> ```
> 楼层 → 累计高度
>   1   →    3 米
>   2   →    6 米
>   3   →    9 米
>   ...
> ```
> 楼层越高，高度越高 → **单调递增**
> 
> **场景 2**：有些层是地下室（负高度）
> ```
> 楼层 → 累计高度
>  -2   →   -6 米
>  -1   →   -3 米
>   0   →    0 米
>   1   →    3 米
> ```
> 仍然是楼层越高，高度越高 → **全程单调递增**
> 
> **场景 3**：塔中间有空中花园（高度不变）
> ```
> 楼层 → 累计高度
>   10  →   30 米
>   11  →   30 米  ← 空中花园
>   12  →   30 米  ← 空中花园
>   13  →   33 米
> ```
> 高度不降低，但可能不变 → **单调不减**
> 
> **信竞中的应用**：单调性是二分查找、三分法、贪心算法的基础！

## 📖 直观理解

### 1. 什么是单调性？

**单调性**描述函数值随自变量变化的**趋势**：

{% mermaid %}
graph TD
    A[单调性] --> B1[单调递增]
    A --> B2[单调递减]
    A --> B3[常数函数]
    
    B1 --> C1["x 增大，f(x) 增大"]
    B2 --> C2["x 增大，f(x) 减小"]
    B3 --> C3["x 增大，f(x) 不变"]
{% endmermaid %}

**图像特征**：
- **单调递增**：从左到右，图像**上升**
- **单调递减**：从左到右，图像**下降**
- **常数函数**：从左到右，图像**水平**

### 2. 严格单调 vs 非严格单调

| 类型 | 定义 | 图像特征 | 示例 |
|------|------|---------|------|
| **严格递增** | x₁ < x₂ → f(x₁) < f(x₂) | 严格上升 | f(x) = x |
| **单调不减** | x₁ < x₂ → f(x₁) ≤ f(x₂) | 上升或水平 | f(x) = ⌊x⌋ |
| **严格递减** | x₁ < x₂ → f(x₁) > f(x₂) | 严格下降 | f(x) = -x |
| **单调不增** | x₁ < x₂ → f(x₁) ≥ f(x₂) | 下降或水平 | f(x) = ⌈-x⌉ |

**示例对比**：

```
严格递增：f(x) = x² (x > 0)
  x:  1  2  3  4
f(x): 1  4  9  16  ✓ 严格增大

单调不减：f(x) = ⌊x⌋ (取整函数)
  x:  1.1  1.9  2.1  2.9
f(x):  1    1    2    2   ✓ 不减，但有相等
```

### 3. 单调区间

函数可能在**不同区间**有不同的单调性：

```
f(x) = x²

在 (-∞, 0] 上：单调递减
在 [0, +∞) 上：单调递增

     y
     |     /
     |    /
     |   /
    _\  /
     \/
     |
-----+----- x
```

**关键点**：x = 0 是**单调性改变**的分界点（极值点）。

## 📐 数学定义

### 单调性的严格定义

设函数 f(x) 的定义域为 I，区间 D ⊆ I：

**1. 单调递增（严格）**

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) < f(x₂)，则称 f(x) 在 D 上**单调递增**。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) < f(x_2)
$$

**2. 单调不减**

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) ≤ f(x₂)，则称 f(x) 在 D 上**单调不减**。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) \leq f(x_2)
$$

**3. 单调递减（严格）**

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) > f(x₂)，则称 f(x) 在 D 上**单调递减**。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) > f(x_2)
$$

**4. 单调不增**

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) ≥ f(x₂)，则称 f(x) 在 D 上**单调不增**。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) \geq f(x_2)
$$

### 单调性的判定方法

**方法 1：定义法**

1. 任取 x₁, x₂ ∈ D，且 x₁ < x₂
2. 计算 f(x₁) - f(x₂) 或 f(x₁)/f(x₂)
3. 判断符号，得出结论

**示例**：证明 f(x) = 2x + 1 在 R 上单调递增

```
证明：
任取 x₁, x₂ ∈ R，且 x₁ < x₂

f(x₁) - f(x₂) = (2x₁ + 1) - (2x₂ + 1)
              = 2(x₁ - x₂)

因为 x₁ < x₂，所以 x₁ - x₂ < 0
所以 2(x₁ - x₂) < 0
即 f(x₁) < f(x₂)

因此 f(x) = 2x + 1 在 R 上单调递增
```

**方法 2：导数法**（后续章节学习）

- f'(x) > 0 → 单调递增
- f'(x) < 0 → 单调递减

**方法 3：图像法**

观察函数图像的升降趋势。

### 常见函数的单调性

| 函数 | 解析式 | 单调区间 |
|------|--------|---------|
| 一次函数 | f(x) = kx + b | k > 0: R 上递增<br>k < 0: R 上递减 |
| 二次函数 | f(x) = ax² + bx + c | a > 0: (-∞, -b/2a] 递减，[-b/2a, +∞) 递增<br>a < 0: 相反 |
| 反比例函数 | f(x) = k/x | k > 0: (-∞, 0) 和 (0, +∞) 递减<br>k < 0: 相反 |
| 指数函数 | f(x) = a^x | a > 1: R 上递增<br>0 < a < 1: R 上递减 |
| 对数函数 | f(x) = logₐ(x) | a > 1: (0, +∞) 上递增<br>0 < a < 1: (0, +∞) 上递减 |

## 💻 代码实现

### 判断函数单调性

```cpp
#include <bits/stdc++.h>
using namespace std;

// 示例函数
int f(int x) {
    return 2 * x + 1;
}

// 判断函数在给定区间上是否单调递增
bool is_increasing(int (*func)(int), int left, int right) {
    for (int x = left; x < right; x++) {
        if (func(x) >= func(x + 1)) {
            return false;
        }
    }
    return true;
}

// 判断函数在给定区间上是否单调递减
bool is_decreasing(int (*func)(int), int left, int right) {
    for (int x = left; x < right; x++) {
        if (func(x) <= func(x + 1)) {
            return false;
        }
    }
    return true;
}

// 判断函数的单调性类型
string check_monotonicity(int (*func)(int), int left, int right) {
    bool inc = is_increasing(func, left, right);
    bool dec = is_decreasing(func, left, right);
    
    if (inc && dec) return "常数函数";
    if (inc) return "单调递增";
    if (dec) return "单调递减";
    return "非单调";
}

int main() {
    cout << check_monotonicity(f, -100, 100) << endl;
    // 输出：单调递增
    
    return 0;
}
```

### 二分查找（利用单调性）

```cpp
// 在单调递增序列中查找目标值
int binary_search(const vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] == target) {
            return mid;  // 找到
        } else if (nums[mid] < target) {
            left = mid + 1;  // 目标在右半部分
        } else {
            right = mid - 1;  // 目标在左半部分
        }
    }
    
    return -1;  // 未找到
}

// 二分查找的变体：查找第一个 >= target 的位置
int lower_bound(const vector<int>& nums, int target) {
    int left = 0, right = nums.size();
    
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    
    return left;
}

// 二分查找的变体：查找第一个 > target 的位置
int upper_bound(const vector<int>& nums, int target) {
    int left = 0, right = nums.size();
    
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] <= target) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    
    return left;
}
```

### 三分法（求单峰函数极值）

```cpp
// 三分法求单峰函数的最大值
// 前提：函数先增后减（单峰）
double ternary_search(double (*f)(double), double left, double right) {
    const double EPS = 1e-9;  // 精度
    
    while (right - left > EPS) {
        double m1 = left + (right - left) / 3;
        double m2 = right - (right - left) / 3;
        
        if (f(m1) < f(m2)) {
            // 最大值在 [m1, right]
            left = m1;
        } else {
            // 最大值在 [left, m2]
            right = m2;
        }
    }
    
    return left;  // 返回极值点
}

// 示例：求 f(x) = -x² + 4x 的最大值
double f(double x) {
    return -x * x + 4 * x;
}

int main() {
    double max_x = ternary_search(f, 0, 4);
    double max_val = f(max_x);
    
    cout << "最大值点：x = " << max_x << endl;
    cout << "最大值：f(x) = " << max_val << endl;
    
    return 0;
}
```

### 单调栈

```cpp
// 单调栈：维护一个单调递增/递减的栈
// 应用：求每个元素左边/右边第一个比它大/小的元素

vector<int> next_greater_element(const vector<int>& nums) {
    int n = nums.size();
    vector<int> result(n, -1);  // 默认 -1（不存在）
    stack<int> stk;  // 存储下标
    
    for (int i = 0; i < n; i++) {
        // 维护单调递减栈
        while (!stk.empty() && nums[i] > nums[stk.top()]) {
            result[stk.top()] = nums[i];
            stk.pop();
        }
        stk.push(i);
    }
    
    return result;
}

// 示例：
// 输入：[2, 1, 2, 4, 3]
// 输出：[2, 2, 4, -1, -1]
// 解释：2 右边第一个比它大的是 2，1 右边第一个比它大的是 2，...
```

## 🏆 信竞应用

### 1. 二分答案（利用单调性）

**核心思想**：如果问题具有单调性，可以二分答案。

```cpp
// 例题：最小的最大边权
// 问题：选择 k 条边，使得最大边权最小

bool check(int max_weight, const vector<int>& weights, int k) {
    // 判断是否能选出 k 条边，权值都不超过 max_weight
    int count = 0;
    for (int w : weights) {
        if (w <= max_weight) {
            count++;
        }
    }
    return count >= k;  // 单调性：max_weight 越大，越容易满足
}

int binary_search_answer(const vector<int>& weights, int k) {
    int left = 0, right = 1e9;
    
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        if (check(mid, weights, k)) {
            right = mid;  // 满足条件，尝试更小的值
        } else {
            left = mid + 1;  // 不满足，需要更大的值
        }
    }
    
    return left;
}
```

### 2. 贪心算法（利用单调性）

**活动选择问题**：

```cpp
struct Activity {
    int start, finish;
};

// 按结束时间排序（单调性）
bool compare_activities(const Activity& a, const Activity& b) {
    return a.finish < b.finish;
}

int max_activities(vector<Activity>& activities) {
    sort(activities.begin(), activities.end(), compare_activities);
    
    int count = 1;
    int last_finish = activities[0].finish;
    
    for (int i = 1; i < activities.size(); i++) {
        // 贪心：选择第一个不冲突的活动
        if (activities[i].start >= last_finish) {
            count++;
            last_finish = activities[i].finish;
        }
    }
    
    return count;
}
```

### 3. 单调队列优化 DP

```cpp
// 滑动窗口最大值
// 问题：求长度为 k 的滑动窗口中的最大值

vector<int> max_sliding_window(const vector<int>& nums, int k) {
    deque<int> dq;  // 存储下标，维护单调递减
    vector<int> result;
    
    for (int i = 0; i < nums.size(); i++) {
        // 移除超出窗口的元素
        while (!dq.empty() && dq.front() <= i - k) {
            dq.pop_front();
        }
        
        // 维护单调递减
        while (!dq.empty() && nums[dq.back()] <= nums[i]) {
            dq.pop_back();
        }
        
        dq.push_back(i);
        
        // 记录结果
        if (i >= k - 1) {
            result.push_back(nums[dq.front()]);
        }
    }
    
    return result;
}
```

### 4. 最长上升子序列（LIS）

```cpp
// O(n²) 解法
int lis_n2(const vector<int>& nums) {
    int n = nums.size();
    vector<int> dp(n, 1);
    
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
    }
    
    return *max_element(dp.begin(), dp.end());
}

// O(n log n) 解法（利用单调性）
int lis_nlogn(const vector<int>& nums) {
    vector<int> tails;  // tails[i] 表示长度为 i+1 的 LIS 的最小结尾元素
    
    for (int x : nums) {
        // 二分查找第一个 >= x 的位置
        auto it = lower_bound(tails.begin(), tails.end(), x);
        
        if (it == tails.end()) {
            tails.push_back(x);  // x 比所有结尾都大
        } else {
            *it = x;  // 替换，保持单调性
        }
    }
    
    return tails.size();
}
```

### 进阶知识（选学）

1. **凸函数与凹函数**
   - 凸函数：f''(x) ≥ 0，图像向下凸
   - 凹函数：f''(x) ≤ 0，图像向上凸
   - 凸函数有唯一极小值

2. **对勾函数**
   - f(x) = x + a/x (a > 0)
   - 在 (0, √a) 递减，在 (√a, +∞) 递增
   - 最小值在 x = √a 处

3. **复合函数的单调性**
   - 同增异减：内外函数单调性相同则增，相反则减
