## 🎯 例题精讲

### 例题 1：函数的定义域

**题目来源**: 课本习题

**题意分析**: 

求下列函数的定义域：

1. $f(x) = \frac{1}{x - 2}$
2. $f(x) = \sqrt{x + 3}$
3. $f(x) = \frac{\sqrt{x - 1}}{x - 5}$

**解题思路**: 

1. 分母不能为零
2. 根号内必须非负
3. 多个条件取交集

**参考答案**:

1. **f(x) = 1/(x - 2)**
   - 条件：x - 2 ≠ 0
   - 解得：x ≠ 2
   - 定义域：{x | x ≠ 2}

2. **f(x) = √(x + 3)**
   - 条件：x + 3 ≥ 0
   - 解得：x ≥ -3
   - 定义域：{x | x ≥ -3}

3. **f(x) = √(x - 1)/(x - 5)**
   - 条件 1：x - 1 ≥ 0 → x ≥ 1
   - 条件 2：x - 5 ≠ 0 → x ≠ 5
   - 取交集：x ≥ 1 且 x ≠ 5
   - 定义域：{x | x ≥ 1 且 x ≠ 5}

### 例题 2：函数值的计算

**题目来源**: 课本习题

**题意分析**: 

已知函数 f(x) = 2x² - 3x + 1，求：

1. f(0)
2. f(2)
3. f(a + 1)
4. f(f(1))

**解题思路**: 

直接代入计算，注意运算顺序。

**参考答案**:

1. **f(0)** = 2(0)² - 3(0) + 1 = 1

2. **f(2)** = 2(2)² - 3(2) + 1 = 8 - 6 + 1 = 3

3. **f(a + 1)** = 2(a + 1)² - 3(a + 1) + 1
   = 2(a² + 2a + 1) - 3a - 3 + 1
   = 2a² + 4a + 2 - 3a - 2
   = 2a² + a

4. **f(f(1))**
   - 先算 f(1) = 2(1)² - 3(1) + 1 = 0
   - 再算 f(f(1)) = f(0) = 1

### 例题 3：分段函数求值

**题目来源**: 自编

**题意分析**: 

已知分段函数：
$$
f(x) = \begin{cases}
x + 1, & x < 0 \\
x^2, & 0 \leq x < 2 \\
2x - 1, & x \geq 2
\end{cases}
$$

求：f(-2), f(0), f(1.5), f(3)

**解题思路**: 

根据 x 的取值范围，选择对应的表达式。

**参考答案**:

- **f(-2)**: x = -2 < 0，用第一段：f(-2) = -2 + 1 = -1
- **f(0)**: x = 0，0 ≤ 0 < 2，用第二段：f(0) = 0² = 0
- **f(1.5)**: x = 1.5，0 ≤ 1.5 < 2，用第二段：f(1.5) = 1.5² = 2.25
- **f(3)**: x = 3 ≥ 2，用第三段：f(3) = 2(3) - 1 = 5

### 例题 4：函数的 C++ 实现

**题目来源**: 洛谷

**题意分析**: 

实现一个函数，计算 f(n) = 1 + 2 + ... + n

**要求**:
1. 用递归实现
2. 用迭代实现
3. 用公式实现

**参考思路**:

```cpp
// 方法 1：递归
int sum_recursive(int n) {
    if (n <= 0) return 0;
    return n + sum_recursive(n - 1);
}

// 方法 2：迭代
int sum_iterative(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
}

// 方法 3：公式（等差数列求和）
int sum_formula(int n) {
    return n * (n + 1) / 2;
}

// 时间复杂度对比：
// 递归：O(n)，可能栈溢出
// 迭代：O(n)，安全
// 公式：O(1)，最优
```

## 📊 本章小结

### 重点回顾

1. **函数的定义**：每个输入对应唯一输出的对应关系
2. **三要素**：定义域、对应法则、值域
3. **表示方法**：解析法、列表法、图像法、代码实现
4. **分段函数**：不同区间有不同表达式
5. **复合函数**：f(g(x))，不满足交换律
6. **C++ 实现**：函数定义、参数传递、返回值

### 易错点提醒

⚠️ **定义域优先**：求函数值前，先确认 x 在定义域内

⚠️ **分段函数**：注意分界点属于哪一段

⚠️ **复合函数顺序**：f(g(x)) ≠ g(f(x))

⚠️ **递归终止条件**：必须有，否则无限递归

⚠️ **函数唯一性**：一个 x 只能对应一个 y
