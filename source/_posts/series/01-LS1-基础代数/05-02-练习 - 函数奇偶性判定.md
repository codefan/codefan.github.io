## 🎯 例题精讲

### 例题 1：判断函数奇偶性

**题目来源**: 课本习题

**题意分析**: 

判断下列函数的奇偶性：

1. f(x) = x⁴
2. f(x) = x³ + x
3. f(x) = x² + x
4. f(x) = 0

**解题思路**: 

根据定义，计算 f(-x) 并与 f(x)、-f(x) 比较。

**参考答案**:

1. **f(x) = x⁴**
   - f(-x) = (-x)⁴ = x⁴ = f(x)
   - **偶函数**

2. **f(x) = x³ + x**
   - f(-x) = (-x)³ + (-x) = -x³ - x = -(x³ + x) = -f(x)
   - **奇函数**

3. **f(x) = x² + x**
   - f(-x) = (-x)² + (-x) = x² - x
   - f(-x) ≠ f(x) 且 f(-x) ≠ -f(x)
   - **非奇非偶函数**

4. **f(x) = 0**
   - f(-x) = 0 = f(x) 且 f(-x) = 0 = -f(x)
   - **既是奇函数又是偶函数**（唯一的特例）

### 例题 2：利用奇偶性求值

**题目来源**: 课本习题

**题意分析**: 

已知 f(x) 是奇函数，且 f(2) = 5，求 f(-2)。

**解题思路**: 

直接利用奇函数定义。

**参考答案**:

```
因为 f(x) 是奇函数
所以 f(-x) = -f(x)

f(-2) = -f(2) = -5
```

### 例题 3：奇偶函数的运算

**题目来源**: 自编

**题意分析**: 

已知 f(x) 是奇函数，g(x) 是偶函数，判断下列函数的奇偶性：

1. h(x) = f(x) + g(x)
2. h(x) = f(x) × g(x)
3. h(x) = g(f(x))

**参考答案**:

1. **h(x) = f(x) + g(x)**
   - h(-x) = f(-x) + g(-x) = -f(x) + g(x)
   - h(-x) ≠ h(x) 且 h(-x) ≠ -h(x)
   - **非奇非偶函数**

2. **h(x) = f(x) × g(x)**
   - h(-x) = f(-x) × g(-x) = -f(x) × g(x) = -h(x)
   - **奇函数**

3. **h(x) = g(f(x))**
   - h(-x) = g(f(-x)) = g(-f(x))
   - 因为 g 是偶函数，g(-f(x)) = g(f(x)) = h(x)
   - **偶函数**

## 📊 本章小结

### 重点回顾

1. **偶函数**：f(-x) = f(x)，关于 y 轴对称
2. **奇函数**：f(-x) = -f(x)，关于原点对称
3. **判定步骤**：先看定义域，再计算 f(-x)
4. **运算性质**：奇 + 奇=奇，偶 + 偶=偶，奇×奇=偶，...
5. **特例**：f(x) = 0 既是奇函数又是偶函数

### 易错点提醒

⚠️ **定义域对称**：定义域不关于原点对称，直接判定为非奇非偶

⚠️ **分段函数**：需要分段验证

⚠️ **复合函数**：奇偶性可能改变

⚠️ **f(x) = 0**：唯一的既是奇函数又是偶函数的函数
