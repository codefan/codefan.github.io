## 🎮 情景引入

> 想象你在照**镜子**：
> 
> **场景 1**：站在平面镜前
> ```
> 你举起右手 → 镜子里的"你"举起左手
> 你向左走   → 镜子里的"你"向右走
> ```
> 左右对称 → **偶函数**（关于 y 轴对称）
> 
> **场景 2**：站在哈哈镜前（中心对称）
> ```
> 你站在 (2, 3)  → 镜子里的"你"在 (-2, -3)
> 你站在 (1, -4) → 镜子里的"你"在 (-1, 4)
> ```
> 中心对称 → **奇函数**（关于原点对称）
> 
> **场景 3**：站在不规则镜子前
> ```
> 没有对称关系 → **非奇非偶函数**
> ```
> 
> **信竞中的应用**：奇偶性可以简化计算、优化算法、快速判断！

## 📖 直观理解

### 1. 什么是奇偶性？

**奇偶性**描述函数图像的**对称性**：

{% mermaid %}
graph TD
    A[奇偶性] --> B1[偶函数]
    A --> B2[奇函数]
    A --> B3[非奇非偶]
    
    B1 --> C1["关于 y 轴对称"]
    B2 --> C2["关于原点对称"]
    B3 --> C3["无对称性"]
{% endmermaid %}

**图像特征**：

| 类型 | 对称性 | 图像特征 | 示例 |
|------|--------|---------|------|
| **偶函数** | 关于 y 轴对称 | 左右镜像 | f(x) = x² |
| **奇函数** | 关于原点对称 | 旋转 180°重合 | f(x) = x³ |
| **非奇非偶** | 无对称 | 不对称 | f(x) = x² + x |

### 2. 偶函数的直观理解

**偶函数**：f(-x) = f(x)

```
x:  -3  -2  -1   0   1   2   3
f(x): 9   4   1   0   1   4   9

规律：f(-x) = f(x)
```

**图像**：
```
    y
    |     /‾‾\
    |    /    \
    |   /      \
____|__/________\____ x
   -2  0       2
```

**生活例子**：
- 抛物线拱桥
- 二次函数的图像

### 3. 奇函数的直观理解

**奇函数**：f(-x) = -f(x)

```
x:  -3  -2  -1   0   1   2   3
f(x): -27 -8  -1   0   1   8   27

规律：f(-x) = -f(x)
```

**图像**：
```
    y
    |    / 
    |   /  
    |  /   
____|_/_______ x
   /| -2   2
  / |
 /  |
```

**生活例子**：
- 三次函数图像
- 正弦函数 y = sin(x)

## 📐 数学定义

### 奇偶性的严格定义

设函数 f(x) 的定义域为 D，且 D 关于原点对称（即 x ∈ D ⇒ -x ∈ D）：

**1. 偶函数**

如果对于 D 上的任意 x，都有 f(-x) = f(x)，则称 f(x) 为**偶函数**。

$$
\forall x \in D, f(-x) = f(x)
$$

**2. 奇函数**

如果对于 D 上的任意 x，都有 f(-x) = -f(x)，则称 f(x) 为**奇函数**。

$$
\forall x \in D, f(-x) = -f(x)
$$

**3. 非奇非偶函数**

如果 f(x) 既不满足偶函数定义，也不满足奇函数定义，则称 f(x) 为**非奇非偶函数**。

### 奇偶性的判定步骤

**步骤 1**：检查定义域是否关于原点对称

```
✓ 对称：R, [-a, a], (-∞, +∞)
✗ 不对称：[0, +∞), [1, 5]
```

如果定义域不对称，**直接判定为非奇非偶**。

**步骤 2**：计算 f(-x)

**步骤 3**：比较 f(-x) 与 f(x)、-f(x) 的关系

- f(-x) = f(x) → 偶函数
- f(-x) = -f(x) → 奇函数
- 都不满足 → 非奇非偶

### 常见函数的奇偶性

| 函数类型 | 解析式 | 奇偶性 | 说明 |
|---------|--------|--------|------|
| 常数函数 | f(x) = c | 偶函数 | f(-x) = c = f(x) |
| 正比例函数 | f(x) = kx | 奇函数 | f(-x) = -kx = -f(x) |
| 一次函数 | f(x) = kx + b | 非奇非偶 (b≠0) | f(-x) ≠ ±f(x) |
| 二次函数 | f(x) = ax² | 偶函数 | f(-x) = a(-x)² = ax² |
| 三次函数 | f(x) = x³ | 奇函数 | f(-x) = (-x)³ = -x³ |
| 幂函数 | f(x) = xⁿ | n 偶：偶函数<br>n 奇：奇函数 | 看指数 |
| 绝对值函数 | f(x) = \|x\| | 偶函数 | \|-x\| = \|x\| |
| 倒数函数 | f(x) = 1/x | 奇函数 | 1/(-x) = -1/x |

### 奇偶函数的运算性质

| 运算 | 奇 + 奇 | 偶 + 偶 | 奇 + 偶 | 奇 × 奇 | 偶 × 偶 | 奇 × 偶 |
|------|-------|-------|-------|-------|-------|-------|
| 结果 | 奇 | 偶 | 非奇非偶 | 偶 | 偶 | 奇 |

**示例**：
```
f(x) = x (奇), g(x) = x² (偶)

f(x) + g(x) = x + x²      → 非奇非偶
f(x) × g(x) = x × x² = x³ → 奇函数
g(x) × g(x) = x² × x² = x⁴ → 偶函数
```

## 💻 代码实现

### 判断函数奇偶性

```cpp
#include <bits/stdc++.h>
using namespace std;

// 示例函数
int f_square(int x) {
    return x * x;  // 偶函数
}

int f_cube(int x) {
    return x * x * x;  // 奇函数
}

int f_linear(int x) {
    return 2 * x + 1;  // 非奇非偶
}

// 判断函数奇偶性
string check_parity(int (*func)(int), int test_range = 100) {
    bool is_even = true;
    bool is_odd = true;
    
    for (int x = 1; x <= test_range; x++) {
        int fx = func(x);
        int f_neg_x = func(-x);
        
        if (f_neg_x != fx) {
            is_even = false;
        }
        if (f_neg_x != -fx) {
            is_odd = false;
        }
        
        // 提前退出
        if (!is_even && !is_odd) {
            return "非奇非偶函数";
        }
    }
    
    if (is_even) return "偶函数";
    if (is_odd) return "奇函数";
    return "非奇非偶函数";
}

int main() {
    cout << "f(x) = x²: " << check_parity(f_square) << endl;
    cout << "f(x) = x³: " << check_parity(f_cube) << endl;
    cout << "f(x) = 2x+1: " << check_parity(f_linear) << endl;
    
    return 0;
}
```

### 利用奇偶性简化计算

```cpp
// 计算对称区间上的函数值和
// 如果 f(x) 是奇函数，则 sum = 0
// 如果 f(x) 是偶函数，则 sum = 2 * sum(0 to n)

// 奇函数：f(x) = x³
long long sum_odd_function(int n) {
    // f(x) = x³ 是奇函数
    // sum(-n to n) = 0
    return 0;  // 直接返回 0，无需计算
}

// 偶函数：f(x) = x²
long long sum_even_function(int n) {
    // f(x) = x² 是偶函数
    // sum(-n to n) = 2 * sum(0 to n)
    long long sum = 0;
    for (int x = 0; x <= n; x++) {
        sum += x * x;
    }
    return 2 * sum - 0;  // 减去重复计算的 f(0)
}

// 一般情况
long long sum_general(int (*func)(int), int n) {
    // 先判断奇偶性
    bool is_even = true;
    bool is_odd = true;
    
    for (int x = 1; x <= min(n, 100); x++) {
        if (func(-x) != func(x)) is_even = false;
        if (func(-x) != -func(x)) is_odd = false;
    }
    
    if (is_odd) {
        return 0;  // 奇函数在对称区间和为 0
    } else if (is_even) {
        long long sum = 0;
        for (int x = 0; x <= n; x++) {
            sum += func(x);
        }
        return 2 * sum - func(0);
    } else {
        // 非奇非偶，直接计算
        long long sum = 0;
        for (int x = -n; x <= n; x++) {
            sum += func(x);
        }
        return sum;
    }
}
```

### 奇偶性在位运算中的应用

```cpp
// 判断一个数的二进制中 1 的个数的奇偶性
bool has_odd_ones(int n) {
    int count = 0;
    while (n > 0) {
        count += n & 1;
        n >>= 1;
    }
    return count % 2 == 1;
}

// 更高效的实现（异或所有位）
bool has_odd_ones_fast(int n) {
    bool parity = false;
    while (n > 0) {
        parity ^= (n & 1);
        n >>= 1;
    }
    return parity;
}

// 使用内置函数（GCC）
bool has_odd_ones_builtin(int n) {
    return __builtin_parity(n);  // 返回 1 的个数的奇偶性
}
```

## 🏆 信竞应用

### 1. 奇偶性优化求和

**问题**：计算 ∑(i=-n to n) f(i)

```cpp
// 如果 f(x) 是奇函数，直接返回 0
// 如果 f(x) 是偶函数，计算一半即可

long long symmetric_sum(int (*f)(int), int n) {
    // 检查奇偶性
    bool is_odd = true;
    for (int x = 1; x <= min(n, 100); x++) {
        if (f(-x) != -f(x)) {
            is_odd = false;
            break;
        }
    }
    
    if (is_odd) {
        return 0;  // 奇函数，对称区间和为 0
    }
    
    // 否则正常计算
    long long sum = 0;
    for (int x = -n; x <= n; x++) {
        sum += f(x);
    }
    return sum;
}
```

### 2. 奇偶性在博弈中的应用

**Nim 游戏**的变种：

```cpp
// 判断先手是否必胜
// 如果石子总数的奇偶性满足某种条件，先手必胜

bool can_first_player_win(const vector<int>& piles) {
    int xor_sum = 0;
    for (int x : piles) {
        xor_sum ^= x;
    }
    
    // Nim 游戏：xor_sum != 0 时先手必胜
    return xor_sum != 0;
}

// 奇偶性应用：某些博弈问题可以简化为奇偶性判断
```

### 3. 奇偶性在图论中的应用

**二分图判定**：

```cpp
// 二分图：顶点可以分为两个集合，每条边的两个端点属于不同集合
// 等价于：图中所有环的长度都是偶数

bool is_bipartite(int n, const vector<vector<int>>& adj) {
    vector<int> color(n, 0);  // 0: 未染色，1: 颜色 A，-1: 颜色 B
    
    function<bool(int, int)> dfs = [&](int u, int c) {
        color[u] = c;
        for (int v : adj[u]) {
            if (color[v] == 0) {
                if (!dfs(v, -c)) return false;
            } else if (color[v] == c) {
                return false;  // 相邻顶点颜色相同
            }
        }
        return true;
    };
    
    for (int i = 0; i < n; i++) {
        if (color[i] == 0) {
            if (!dfs(i, 1)) return false;
        }
    }
    
    return true;
}
```

### 4. 排列的奇偶性

```cpp
// 排列的奇偶性：交换次数的奇偶性
// 应用：八数码问题、拼图游戏可解性判定

int count_inversions(const vector<int>& a) {
    int n = a.size();
    int inversions = 0;
    
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            if (a[i] > a[j]) {
                inversions++;
            }
        }
    }
    
    return inversions;
}

bool is_even_permutation(const vector<int>& a) {
    return count_inversions(a) % 2 == 0;
}

// 应用：判断八数码问题是否有解
// 如果逆序对数为偶数，则有解
```

## 📚 拓展阅读

### 进阶知识（选学）

1. **奇偶分解**
   - 任何函数都可以分解为奇函数 + 偶函数
   - f(x) = [f(x) + f(-x)]/2 + [f(x) - f(-x)]/2

2. **周期函数的奇偶性**
   - sin(x)：奇函数，周期 2π
   - cos(x)：偶函数，周期 2π

3. **傅里叶级数**
   - 奇函数的傅里叶级数只含正弦项
   - 偶函数的傅里叶级数只含余弦项
