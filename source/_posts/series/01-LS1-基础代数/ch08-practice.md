## 🎯 例题精讲

### 例题 1：幂函数的性质

**题目来源**: 课本习题

**题意分析**: 

比较下列各组数的大小：
1. 2³ 和 3²
2. 2^0.5 和 2^0.3
3. (-2)³ 和 (-2)²

**参考答案**:

1. 2³ = 8, 3² = 9，所以 2³ < 3²
2. 因为 y = 2^x 在 R 上递增，0.5 > 0.3，所以 2^0.5 > 2^0.3
3. (-2)³ = -8, (-2)² = 4，所以 (-2)³ < (-2)²

---

### 例题 2：对勾函数最值

**题目来源**: 自编

**题意分析**: 

求函数 f(x) = x + 4/x (x > 0) 的最小值。

**解题思路**: 

利用对勾函数性质或基本不等式。

**参考答案**:

```
方法 1：对勾函数性质
最小值在 x = √4 = 2 处取得
f(2) = 2 + 4/2 = 4

方法 2：基本不等式
x + 4/x ≥ 2√(x · 4/x) = 2√4 = 4
等号当 x = 4/x，即 x = 2 时成立

答案：最小值为 4
```

## 📊 本章小结

### 重点回顾

1. **幂函数**：f(x) = x^a，a 为常数
2. **对勾函数**：f(x) = x + a/x，最小值 2√a
3. **基本不等式**：a + b ≥ 2√(ab)
4. **快速幂**：O(log n) 计算幂

### 易错点提醒

⚠️ **负指数**：x^(-n) = 1/x^n

⚠️ **定义域**：对勾函数 x ≠ 0

⚠️ **等号条件**：基本不等式等号当且仅当 a = b
