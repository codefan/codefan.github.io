## 🎯 例题精讲

### 例题 1：集合的交集与并集

**题目来源**: 洛谷

**题意分析**: 

给定两个集合 A 和 B，每个集合包含 n 个整数。请计算：
1. A ∩ B（交集）
2. A ∪ B（并集）

**输入格式**:
```
第一行：n (1 ≤ n ≤ 10^5)
第二行：n 个整数，表示集合 A
第三行：n 个整数，表示集合 B
```

**输出格式**:
```
第一行：交集中元素的个数
第二行：并集中元素的个数
```

**解题思路**: 

{% mermaid %}
graph LR
    A[读入数据] --> B[存入 set 去重]
    B --> C[计算交集]
    B --> D[计算并集]
    C --> E[输出结果]
    D --> E
{% endmermaid %}


1. 使用 C++ STL 的 `set` 自动去重
2. 用 `set_intersection` 和 `set_union` 计算交集和并集
3. 注意：结果也需要去重

**参考思路**:
```cpp
// 提示：使用 STL set 和算法库函数
// set_intersection 计算交集
// set_union 计算并集
// 时间复杂度：O(n log n)
```

### 例题 3：状态压缩入门

**题目来源**: 洛谷

**题意分析**: 

有 n 块奶酪，老鼠从原点出发，要吃掉所有奶酪。问最短路径是多少？

**数据范围**: n ≤ 15（提示：状态压缩）

**解题思路**: 

1. 用二进制位表示"已吃过的奶酪集合"
2. `dp[mask][i]` = 吃过 mask 表示的奶酪集合，最后在奶酪 i 的最短路径
3. 状态转移：枚举下一个要吃的奶酪 j

**参考思路**:
```cpp

// 2. dp[mask][i] 表示状态
```

### 例题 4：容斥原理应用

**题目来源**: Codeforces

**题意分析**: 

求 1~n 中能被 a 或 b 或 c 整除的数的个数。

**输入格式**:
```
一行四个整数：n, a, b, c (1 ≤ n ≤ 10^9, 1 ≤ a,b,c ≤ 10^9)
```

**输出格式**:
```
一个整数，表示答案
```

**解题思路**: 

使用容斥原理：
- 能被 a 整除的数的个数：⌊n/a⌋
- 能被 a 和 b 同时整除的数的个数：⌊n/lcm(a,b)⌋
- 应用容斥原理公式

## 📊 本章小结

### 重点回顾

1. **集合三特性**：确定性、互异性、无序性
2. **四种运算**：
   - 交集：A ∩ B（共有元素）
   - 并集：A ∪ B（所有元素，去重）
   - 补集：∁ᵤA（不属于 A 的元素）
   - 差集：A - B（在 A 中但不在 B 中）
3. **集合运算律**：交换律、结合律、分配律、德摩根定律
4. **容斥原理**：计算多个集合并集的大小
5. **逻辑联结词**：∧（且）、∨（或）、¬（非）、→（蕴含）、↔（等价）
6. **量词**：∀（全称）、∃（存在）、∃!（唯一存在）
7. **位运算表示集合**：状态压缩的核心思想

### 易错点提醒

⚠️ **集合并集要去重**：{1,2,3} ∪ {2,3,4} = {1,2,3,4}（不是 {1,2,3,2,3,4}）

⚠️ **空集是任何集合的子集**：∅ ⊆ A 恒成立

⚠️ **量词否定要变号**：
- "所有人都会" 的否定是 "有人不会"（不是 "所有人都不会"）
- "有人会" 的否定是 "所有人都不会"

⚠️ **蕴含的真值表**：p→q 只有"前真后假"时为假

⚠️ **量词顺序不能交换**：∀x∃y 和 ∃y∀x 不等价

### 信竞应用总结

| 知识点 | 应用场景 | 典型题目 |
|--------|---------|---------|
| 集合运算 | 数据去重、统计 | STL set |
| 位运算集合 | 状态压缩 DP | 旅行商问题 |
| 容斥原理 | 计数问题 | 整除计数 |
| 逻辑判断 | 条件分支、博弈 | AtCoder ABC 系列 |
| 德摩根定律 | 条件优化、剪枝 | 搜索优化 |
