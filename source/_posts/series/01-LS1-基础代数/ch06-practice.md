## 🎯 例题精讲

### 例题 1：平移变换

**题目来源**: 课本习题

**题意分析**: 

将函数 y = x² 的图像向右平移 2 个单位，再向上平移 3 个单位，求变换后的函数解析式。

**解题思路**: 

根据平移规则：
- 向右平移 2：x → x - 2
- 向上平移 3：y → y - 3

**参考答案**:

```
原函数：y = x²

向右平移 2：y = (x - 2)²
向上平移 3：y = (x - 2)² + 3

答案：y = (x - 2)² + 3
```

### 例题 2：对称变换

**题目来源**: 课本习题

**题意分析**: 

已知函数 f(x) = 2x + 1，求：
1. 关于 y 轴对称的函数
2. 关于 x 轴对称的函数
3. 关于原点对称的函数

**解题思路**: 

根据对称变换规则：
- 关于 y 轴对称：f(x) → f(-x)
- 关于 x 轴对称：f(x) → -f(x)
- 关于原点对称：f(x) → -f(-x)

**参考答案**:

1. **关于 y 轴对称**：
   - f(-x) = 2(-x) + 1 = -2x + 1

2. **关于 x 轴对称**：
   - -f(x) = -(2x + 1) = -2x - 1

3. **关于原点对称**：
   - -f(-x) = -(-2x + 1) = 2x - 1

### 例题 3：伸缩变换

**题目来源**: 自编

**题意分析**: 

函数 y = sin(x) 经过怎样的变换可以得到 y = 2sin(3x)？

**解题思路**: 

对比系数：
- 振幅：1 → 2（纵向拉伸 2 倍）
- 频率：1 → 3（横向压缩为 1/3）

**参考答案**:

```
变换步骤：
1. 横向压缩为原来的 1/3：y = sin(3x)
2. 纵向拉伸 2 倍：y = 2sin(3x)

或者：
1. 纵向拉伸 2 倍：y = 2sin(x)
2. 横向压缩为原来的 1/3：y = 2sin(3x)
```

## 📊 本章小结

### 重点回顾

1. **平移变换**：左加右减，上加下减
2. **对称变换**：y 轴对称 f(-x)，x 轴对称 -f(x)，原点对称 -f(-x)
3. **伸缩变换**：横向 f(ωx)，纵向 Af(x)
4. **复合变换**：注意顺序，一般形式 y = A·f(ωx+φ)+h

### 易错点提醒

⚠️ **平移方向**：f(x-a) 是向右平移 a，不是向左

⚠️ **变换顺序**：先伸缩后平移，还是先平移后伸缩，结果不同

⚠️ **符号处理**：关于原点对称是 -f(-x)，不是 -f(x)

### 信竞应用总结

| 知识点 | 应用场景 | 典型题目 |
|--------|---------|---------|
| 坐标平移 | 几何变换 | 图形移动 |
| 对称变换 | 镜像问题 | 对称图形 |
| 伸缩变换 | 缩放问题 | 地图缩放 |
| 二次函数 | 抛物线 | 物理模拟 |
| 三角函数 | 波动问题 | 正弦波 |
