例题 2：对称变换

题目来源: 课本习题

题意分析:

已知函数 f(x) = 2x + 1，求：

  1. 关于 y 轴对称的函数
  2. 关于 x 轴对称的函数
  3. 关于原点对称的函数

解题思路:

根据对称变换规则：

  • 关于 y 轴对称：f(x) → f(-x)
  • 关于 x 轴对称：f(x) → -f(x)
  • 关于原点对称：f(x) → -f(-x)

参考答案:

  1. 关于 y 轴对称：

    • f(-x) = 2(-x) + 1 = -2x + 1

  2. 关于 x 轴对称：

    • -f(x) = -(2x + 1) = -2x - 1

  3. 关于原点对称：

    • -f(-x) = -(-2x + 1) = 2x - 1

例题 1：判断函数奇偶性

题目来源: 课本习题

题意分析:

判断下列函数的奇偶性：

  1. f(x) = x⁴
  2. f(x) = x³ + x
  3. f(x) = x² + x
  4. f(x) = 0

解题思路:

根据定义，计算 f(-x) 并与 f(x)、-f(x) 比较。

参考答案:

  1. f(x) = x⁴

    • f(-x) = (-x)⁴ = x⁴ = f(x)
    • 偶函数

  2. f(x) = x³ + x

    • f(-x) = (-x)³ + (-x) = -x³ - x = -(x³ + x) = -f(x)
    • 奇函数

  3. f(x) = x² + x

    • f(-x) = (-x)² + (-x) = x² - x
    • f(-x) ≠ f(x) 且 f(-x) ≠ -f(x)
    • 非奇非偶函数

  4. f(x) = 0

    • f(-x) = 0 = f(x) 且 f(-x) = 0 = -f(x)
    • 既是奇函数又是偶函数（唯一的特例）

例题 3：奇偶函数的运算

题目来源: 自编

题意分析:

已知 f(x) 是奇函数，g(x) 是偶函数，判断下列函数的奇偶性：

  1. h(x) = f(x) + g(x)
  2. h(x) = f(x) × g(x)
  3. h(x) = g(f(x))

参考答案:

  1. h(x) = f(x) + g(x)

    • h(-x) = f(-x) + g(-x) = -f(x) + g(x)
    • h(-x) ≠ h(x) 且 h(-x) ≠ -h(x)
    • 非奇非偶函数

  2. h(x) = f(x) × g(x)

    • h(-x) = f(-x) × g(-x) = -f(x) × g(x) = -h(x)
    • 奇函数

  3. h(x) = g(f(x))

    • h(-x) = g(f(-x)) = g(-f(x))
    • 因为 g 是偶函数，g(-f(x)) = g(f(x)) = h(x)
    • 偶函数

📐 数学定义

奇偶性的严格定义

设函数 f(x) 的定义域为 D，且 D 关于原点对称（即 x ∈ D ⇒ -x ∈ D）：

1. 偶函数

如果对于 D 上的任意 x，都有 f(-x) = f(x)，则称 f(x) 为偶函数。

$$
\forall x \in D, f(-x) = f(x)
$$

2. 奇函数

如果对于 D 上的任意 x，都有 f(-x) = -f(x)，则称 f(x) 为奇函数。

$$
\forall x \in D, f(-x) = -f(x)
$$

3. 非奇非偶函数

如果 f(x) 既不满足偶函数定义，也不满足奇函数定义，则称 f(x) 为非奇非偶函数。

奇偶性的判定步骤

步骤 1：检查定义域是否关于原点对称

如果定义域不对称，直接判定为非奇非偶。

步骤 2：计算 f(-x)

步骤 3：比较 f(-x) 与 f(x)、-f(x) 的关系

  • f(-x) = f(x) → 偶函数
  • f(-x) = -f(x) → 奇函数
  • 都不满足 → 非奇非偶

常见函数的奇偶性

函数类型 解析式 奇偶性 说明
常数函数 f(x) = c 偶函数 f(-x) = c = f(x)
正比例函数 f(x) = kx 奇函数 f(-x) = -kx = -f(x)
一次函数 f(x) = kx + b 非奇非偶 (b≠0) f(-x) ≠ ±f(x)
二次函数 f(x) = ax² 偶函数 f(-x) = a(-x)² = ax²
三次函数 f(x) = x³ 奇函数 f(-x) = (-x)³ = -x³
幂函数 f(x) = xⁿ n 偶：偶函数
n 奇：奇函数		 看指数
绝对值函数 f(x) = |x| 偶函数 |-x| = |x|
倒数函数 f(x) = 1/x 奇函数 1/(-x) = -1/x

运算 奇 + 奇 偶 + 偶 奇 + 偶 奇 × 奇 偶 × 偶 奇 × 偶

[教程] 04-函数的单调性

🎮 情景引入

想象你在玩一个爬塔游戏，塔有 100 层：

场景 1：每层高度相同

楼层越高，高度越高 → 单调递增

场景 2：有些层是地下室（负高度）

仍然是楼层越高，高度越高 → 全程单调递增

场景 3：塔中间有空中花园（高度不变）

高度不降低，但可能不变 → 单调不减

信竞中的应用：单调性是二分查找、三分法、贪心算法的基础！

📖 直观理解

1. 什么是单调性？

单调性描述函数值随自变量变化的趋势：

graph TD
A[单调性] –> B1[单调递增]
A –> B2[单调递减]
A –> B3[常数函数]

B1 --> C1["x 增大，f(x) 增大"]
B2 --> C2["x 增大，f(x) 减小"]
B3 --> C3["x 增大，f(x) 不变"]

图像特征：

  • 单调递增：从左到右，图像上升
  • 单调递减：从左到右，图像下降
  • 常数函数：从左到右，图像水平

2. 严格单调 vs 非严格单调

类型 定义 图像特征 示例
严格递增 x₁ < x₂ → f(x₁) < f(x₂) 严格上升 f(x) = x
单调不减 x₁ < x₂ → f(x₁) ≤ f(x₂) 上升或水平 f(x) = ⌊x⌋
严格递减 x₁ < x₂ → f(x₁) > f(x₂) 严格下降 f(x) = -x
单调不增 x₁ < x₂ → f(x₁) ≥ f(x₂) 下降或水平 f(x) = ⌈-x⌉

📐 数学定义

单调性的严格定义

设函数 f(x) 的定义域为 I，区间 D ⊆ I：

1. 单调递增（严格）

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) < f(x₂)，则称 f(x) 在 D 上单调递增。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) < f(x_2)
$$

2. 单调不减

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) ≤ f(x₂)，则称 f(x) 在 D 上单调不减。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) \leq f(x_2)
$$

3. 单调递减（严格）

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) > f(x₂)，则称 f(x) 在 D 上单调递减。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) > f(x_2)
$$

4. 单调不增

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) ≥ f(x₂)，则称 f(x) 在 D 上单调不增。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) \geq f(x_2)
$$

单调性的判定方法

方法 1：定义法

  1. 任取 x₁, x₂ ∈ D，且 x₁ < x₂
  2. 计算 f(x₁) - f(x₂) 或 f(x₁)/f(x₂)
  3. 判断符号，得出结论

示例：证明 f(x) = 2x + 1 在 R 上单调递增

方法 2：导数法（后续章节学习）

  • f’(x) > 0 → 单调递增
  • f’(x) < 0 → 单调递减

方法 3：图像法

观察函数图像的升降趋势。

常见函数的单调性

函数 解析式 单调区间
一次函数 f(x) = kx + b k > 0: R 上递增
k < 0: R 上递减
二次函数 f(x) = ax² + bx + c a > 0: (-∞, -b/2a] 递减，[-b/2a, +∞) 递增
a < 0: 相反
反比例函数 f(x) = k/x k > 0: (-∞, 0) 和 (0, +∞) 递减
k < 0: 相反
指数函数 f(x) = a^x a > 1: R 上递增
0 < a < 1: R 上递减
对数函数 f(x) = logₐ(x) a > 1: (0, +∞) 上递增
0 < a < 1: (0, +∞) 上递减

例题 1：函数的定义域

题目来源: 课本习题

题意分析:

求下列函数的定义域：

  1. $f(x) = \frac{1}{x - 2}$
  2. $f(x) = \sqrt{x + 3}$
  3. $f(x) = \frac{\sqrt{x - 1}}{x - 5}$

解题思路:

  1. 分母不能为零
  2. 根号内必须非负
  3. 多个条件取交集

参考答案:

  1. f(x) = 1/(x - 2)

    • 条件：x - 2 ≠ 0
    • 解得：x ≠ 2
    • 定义域：{x | x ≠ 2}

  2. f(x) = √(x + 3)

    • 条件：x + 3 ≥ 0
    • 解得：x ≥ -3
    • 定义域：{x | x ≥ -3}

  3. f(x) = √(x - 1)/(x - 5)

    • 条件 1：x - 1 ≥ 0 → x ≥ 1
    • 条件 2：x - 5 ≠ 0 → x ≠ 5
    • 取交集：x ≥ 1 且 x ≠ 5
    • 定义域：{x | x ≥ 1 且 x ≠ 5}

例题 2：函数值的计算

题目来源: 课本习题

题意分析:

已知函数 f(x) = 2x² - 3x + 1，求：

  1. f(0)
  2. f(2)
  3. f(a + 1)
  4. f(f(1))

解题思路:

直接代入计算，注意运算顺序。

参考答案:

  1. f(0) = 2(0)² - 3(0) + 1 = 1

  2. f(2) = 2(2)² - 3(2) + 1 = 8 - 6 + 1 = 3

  3. f(a + 1) = 2(a + 1)² - 3(a + 1) + 1
    = 2(a² + 2a + 1) - 3a - 3 + 1
    = 2a² + 4a + 2 - 3a - 2
    = 2a² + a

  4. f(f(1))

    • 先算 f(1) = 2(1)² - 3(1) + 1 = 0
    • 再算 f(f(1)) = f(0) = 1

例题 3：分段函数求值

题目来源: 自编

题意分析:

已知分段函数：
$$
f(x) = \begin{cases}
x + 1, & x < 0 \
x^2, & 0 \leq x < 2 \
2x - 1, & x \geq 2
\end{cases}
$$

求：f(-2), f(0), f(1.5), f(3)

解题思路:

根据 x 的取值范围，选择对应的表达式。

参考答案:

  • f(-2): x = -2 < 0，用第一段：f(-2) = -2 + 1 = -1
  • f(0): x = 0，0 ≤ 0 < 2，用第二段：f(0) = 0² = 0
  • f(1.5): x = 1.5，0 ≤ 1.5 < 2，用第二段：f(1.5) = 1.5² = 2.25
  • f(3): x = 3 ≥ 2，用第三段：f(3) = 2(3) - 1 = 5

例题 1：充要条件的判断

题目来源: 课本习题

题意分析:

判断下列各题中 p 是 q 的什么条件：

  1. p: x > 2, q: x > 0
  2. p: x² > 4, q: x > 2
  3. p: 三角形是等边三角形，q: 三角形是等角三角形
  4. p: a 是偶数，q: a² 是偶数

解题思路:

对每个小题：

  1. 判断 p → q 是否为真
  2. 判断 q → p 是否为真
  3. 根据真值表得出结论

参考答案:

  1. p: x > 2, q: x > 0

    • p → q: 若 x > 2，则 x > 0 ✅（真）
    • q → p: 若 x > 0，则 x > 2 ❌（假，如 x=1）
    • 结论: 充分不必要条件

  2. p: x² > 4, q: x > 2

    • p → q: 若 x² > 4，则 x > 2 ❌（假，如 x=-3）
    • q → p: 若 x > 2，则 x² > 4 ✅（真）
    • 结论: 必要不充分条件

  3. p: 等边三角形，q: 等角三角形

    • p → q: 等边 → 等角 ✅（真）
    • q → p: 等角 → 等边 ✅（真）
    • 结论: 充要条件

  4. p: a 是偶数，q: a² 是偶数

    • p → q: 偶数 → 平方是偶数 ✅（真）
    • q → p: 平方是偶数 → 原数是偶数 ✅（真，奇数的平方是奇数）
    • 结论: 充要条件

🎮 情景引入

想象你在玩一个解谜游戏，要打开一扇魔法门需要满足条件：

场景 1：门上有块牌子写着”如果有钥匙，就能开门”

  • 你有钥匙 → 一定能开门 ✅
  • 但你没钥匙，门也可能开着（别人开的）❓

场景 2：牌子写着”只有有钥匙，才能开门”

  • 门开着 → 你一定有钥匙 ✅
  • 但你有钥匙，门也可能锁着（你不想开）❓

场景 3：牌子写着”当且仅当有钥匙，门才开着”

  • 有钥匙 ↔ 门开着（完全等价）✅

这就是充分条件、必要条件和充要条件的区别！

在信竞中，判断条件的性质至关重要——什么时候可以剪枝？什么时候可以贪心？什么时候必须搜索？

📖 直观理解

1. 三种条件的含义

充分条件：”有它就够了”

必要条件：”没它可不行”

充要条件：”有且仅有它”

2. 集合视角理解

用集合的观点看条件关系：

集合解释：

  • 若 A ⊆ B，则 “x∈A” 是 “x∈B” 的充分条件
  • 若 B ⊆ A，则 “x∈A” 是 “x∈B” 的必要条件
  • 若 A = B，则 “x∈A” 是 “x∈B” 的充要条件

3. 逻辑符号表示

关系 符号 含义 示例
充分条件 p → q 若 p 则 q 下雨 → 地湿
必要条件 q → p 只有 p 才 q 考驾照 → 年满 18 岁

📐 数学定义

定义

设 p、q 是两个命题：

1. 充分条件

如果 p → q 为真，则称 p 是 q 的充分条件。

$$
p \Rightarrow q \iff \text{p 是 q 的充分条件}
$$

2. 必要条件

如果 q → p 为真，则称 p 是 q 的必要条件。

$$
q \Rightarrow p \iff \text{p 是 q 的必要条件}
$$

3. 充要条件

如果 p → q 且 q → p 都为真，则称 p 是 q 的充要条件（充分必要条件）。

$$
p \Leftrightarrow q \iff \text{p 是 q 的充要条件}
$$

判定方法

方法 1：定义法

  1. 判断 p → q 是否为真
  2. 判断 q → p 是否为真
  3. 得出结论

方法 2：集合法

  1. 写出 p、q 对应的集合 A、B
  2. 判断集合关系：
    • A ⊆ B → 充分条件
    • B ⊆ A → 必要条件
    • A = B → 充要条件

进阶知识（选学）

  1. 逆命题、否命题、逆否命题

    • 原命题：p → q
    • 逆命题：q → p
    • 否命题：¬p → ¬q
    • 逆否命题：¬q → ¬p
    • 重要：原命题与逆否命题等价

  2. 反证法

    • 假设结论不成立，推出矛盾
    • 本质是证明逆否命题

  3. 数学归纳法

    • 证明 n=1 成立
    • 假设 n=k 成立，证明 n=k+1 成立
    • 充要条件：归纳基础 + 归纳步骤
