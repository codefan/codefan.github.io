📐 数学定义

指数运算法则

设 a > 0, b > 0, m, n ∈ R：

法则 公式 示例
乘法 aᵐ × aⁿ = aᵐ⁺ⁿ 2³ × 2⁴ = 2⁷
除法 aᵐ ÷ aⁿ = aᵐ⁻ⁿ 2⁵ ÷ 2² = 2³
幂的幂 (aᵐ)ⁿ = aᵐⁿ (2³)² = 2⁶
积的幂 (ab)ⁿ = aⁿbⁿ (2×3)² = 2²×3²
商的幂 (a/b)ⁿ = aⁿ/bⁿ (4/2)³ = 4³/2³

指数函数的性质

性质 a > 1 0 < a < 1
单调性 单调递增 单调递减
值域 (0, +∞) (0, +∞)
定点 (0, 1) (0, 1)
渐近线 y = 0 (x → -∞) y = 0 (x → +∞)
示例 y = 2^x y = (1/2)^x
