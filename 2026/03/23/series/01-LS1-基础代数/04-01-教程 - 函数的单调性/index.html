[教程] 04-函数的单调性

🎮 情景引入

想象你在玩一个爬塔游戏，塔有 100 层：

场景 1：每层高度相同

楼层越高，高度越高 → 单调递增

场景 2：有些层是地下室（负高度）

仍然是楼层越高，高度越高 → 全程单调递增

场景 3：塔中间有空中花园（高度不变）

高度不降低，但可能不变 → 单调不减

信竞中的应用：单调性是二分查找、三分法、贪心算法的基础！

📖 直观理解

1. 什么是单调性？

单调性描述函数值随自变量变化的趋势：

graph TD
A[单调性] –> B1[单调递增]
A –> B2[单调递减]
A –> B3[常数函数]

B1 --> C1["x 增大，f(x) 增大"]
B2 --> C2["x 增大，f(x) 减小"]
B3 --> C3["x 增大，f(x) 不变"]

图像特征：

  • 单调递增：从左到右，图像上升
  • 单调递减：从左到右，图像下降
  • 常数函数：从左到右，图像水平

2. 严格单调 vs 非严格单调

类型 定义 图像特征 示例
严格递增 x₁ < x₂ → f(x₁) < f(x₂) 严格上升 f(x) = x
单调不减 x₁ < x₂ → f(x₁) ≤ f(x₂) 上升或水平 f(x) = ⌊x⌋
严格递减 x₁ < x₂ → f(x₁) > f(x₂) 严格下降 f(x) = -x
单调不增 x₁ < x₂ → f(x₁) ≥ f(x₂) 下降或水平 f(x) = ⌈-x⌉

📐 数学定义

单调性的严格定义

设函数 f(x) 的定义域为 I，区间 D ⊆ I：

1. 单调递增（严格）

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) < f(x₂)，则称 f(x) 在 D 上单调递增。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) < f(x_2)
$$

2. 单调不减

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) ≤ f(x₂)，则称 f(x) 在 D 上单调不减。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) \leq f(x_2)
$$

3. 单调递减（严格）

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) > f(x₂)，则称 f(x) 在 D 上单调递减。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) > f(x_2)
$$

4. 单调不增

如果对于 D 上任意两个值 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) ≥ f(x₂)，则称 f(x) 在 D 上单调不增。

$$
\forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) \geq f(x_2)
$$

单调性的判定方法

方法 1：定义法

  1. 任取 x₁, x₂ ∈ D，且 x₁ < x₂
  2. 计算 f(x₁) - f(x₂) 或 f(x₁)/f(x₂)
  3. 判断符号，得出结论

示例：证明 f(x) = 2x + 1 在 R 上单调递增

方法 2：导数法（后续章节学习）

  • f’(x) > 0 → 单调递增
  • f’(x) < 0 → 单调递减

方法 3：图像法

观察函数图像的升降趋势。

常见函数的单调性

函数 解析式 单调区间
一次函数 f(x) = kx + b k > 0: R 上递增
k < 0: R 上递减
二次函数 f(x) = ax² + bx + c a > 0: (-∞, -b/2a] 递减，[-b/2a, +∞) 递增
a < 0: 相反
反比例函数 f(x) = k/x k > 0: (-∞, 0) 和 (0, +∞) 递减
k < 0: 相反
指数函数 f(x) = a^x a > 1: R 上递增
0 < a < 1: R 上递减
对数函数 f(x) = logₐ(x) a > 1: (0, +∞) 上递增
0 < a < 1: (0, +∞) 上递减

📚 拓展阅读

进阶知识（选学）

  1. 凸函数与凹函数

    • 凸函数：f’’(x) ≥ 0，图像向下凸
    • 凹函数：f’’(x) ≤ 0，图像向上凸
    • 凸函数有唯一极小值

  2. 对勾函数

    • f(x) = x + a/x (a > 0)
    • 在 (0, √a) 递减，在 (√a, +∞) 递增
    • 最小值在 x = √a 处

  3. 复合函数的单调性

    • 同增异减：内外函数单调性相同则增，相反则减
