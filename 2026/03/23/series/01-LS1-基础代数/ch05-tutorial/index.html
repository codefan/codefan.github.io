[教程] 05-函数的奇偶性

🎮 情景引入

想象你在照镜子：

场景 1：站在平面镜前

左右对称 → 偶函数（关于 y 轴对称）

场景 2：站在哈哈镜前（中心对称）

中心对称 → 奇函数（关于原点对称）

场景 3：站在不规则镜子前

信竞中的应用：奇偶性可以简化计算、优化算法、快速判断！

📖 直观理解

1. 什么是奇偶性？

奇偶性描述函数图像的对称性：

📐 数学定义

奇偶性的严格定义

设函数 f(x) 的定义域为 D，且 D 关于原点对称（即 x ∈ D ⇒ -x ∈ D）：

1. 偶函数

如果对于 D 上的任意 x，都有 f(-x) = f(x)，则称 f(x) 为偶函数。

$$
\forall x \in D, f(-x) = f(x)
$$

2. 奇函数

如果对于 D 上的任意 x，都有 f(-x) = -f(x)，则称 f(x) 为奇函数。

$$
\forall x \in D, f(-x) = -f(x)
$$

3. 非奇非偶函数

如果 f(x) 既不满足偶函数定义，也不满足奇函数定义，则称 f(x) 为非奇非偶函数。

奇偶性的判定步骤

步骤 1：检查定义域是否关于原点对称

如果定义域不对称，直接判定为非奇非偶。

步骤 2：计算 f(-x)

步骤 3：比较 f(-x) 与 f(x)、-f(x) 的关系

  • f(-x) = f(x) → 偶函数
  • f(-x) = -f(x) → 奇函数
  • 都不满足 → 非奇非偶

常见函数的奇偶性

函数类型 解析式 奇偶性 说明
常数函数 f(x) = c 偶函数 f(-x) = c = f(x)
正比例函数 f(x) = kx 奇函数 f(-x) = -kx = -f(x)
一次函数 f(x) = kx + b 非奇非偶 (b≠0) f(-x) ≠ ±f(x)
二次函数 f(x) = ax² 偶函数 f(-x) = a(-x)² = ax²
三次函数 f(x) = x³ 奇函数 f(-x) = (-x)³ = -x³
幂函数 f(x) = xⁿ n 偶：偶函数
n 奇：奇函数		 看指数
绝对值函数 f(x) = |x| 偶函数 |-x| = |x|
倒数函数 f(x) = 1/x 奇函数 1/(-x) = -1/x

奇偶函数的运算性质

运算 奇 + 奇 偶 + 偶 奇 + 偶 奇 × 奇 偶 × 偶 奇 × 偶
结果 奇 偶 非奇非偶 偶 偶 奇

📚 拓展阅读

进阶知识（选学）

  1. 奇偶分解

    • 任何函数都可以分解为奇函数 + 偶函数
    • f(x) = [f(x) + f(-x)]/2 + [f(x) - f(-x)]/2

  2. 周期函数的奇偶性

    • sin(x)：奇函数，周期 2π
    • cos(x)：偶函数，周期 2π

  3. 傅里叶级数

    • 奇函数的傅里叶级数只含正弦项
    • 偶函数的傅里叶级数只含余弦项
