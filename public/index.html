📚 信息竞赛数学教程

面向初中升高一学生的信竞数学基础教程

📋 课程介绍

本教程专为信息学竞赛（OI/CP）选手设计，帮助初中生掌握高中数学核心内容，并理解其在编程竞赛中的应用。

🎯 目标读者

  • 初中升高一学生
  • 有一定编程基础（C++）
  • 准备参加 NOIP/信息学竞赛
  • 希望提高数学水平

📖 课程特色

  1. 循序渐进 - 从初中数学平滑过渡到高中数学
  2. 通俗易懂 - 游戏/竞赛场景引入，避免抽象定义
  3. 代码实现 - 每个知识点都有 C++ 代码示例
  4. 信竞应用 - 强调数学在算法竞赛中的实际应用
  5. 习题丰富 - 每章配备数学题 + 编程题

[教程] 16-三角函数的图像与性质

📖 三角函数的图像

y = sin x

  • 定义域：R
  • 值域：[-1, 1]
  • 周期：2π
  • 奇函数
  • 图像：波浪形

y = cos x

  • 定义域：R
  • 值域：[-1, 1]
  • 周期：2π
  • 偶函数
  • 图像：波浪形（相位差 π/2）

y = tan x

  • 定义域：x ≠ π/2 + kπ
  • 值域：R
  • 周期：π
  • 奇函数
  • 图像：有渐近线

📐 性质总结

函数 定义域 值域 周期 奇偶性
sin x R [-1, 1] 2π 奇
cos x R [-1, 1] 2π 偶
tan x x ≠ π/2+kπ R π 奇

[教程] 14-三角函数的概念

🎮 情景引入

想象你在玩一个旋转游戏：

一个点绕原点旋转，坐标如何变化？

这就是三角函数：x = cos(θ), y = sin(θ)

📖 定义

单位圆定义

在单位圆上，角度θ对应的点坐标为 (cos θ, sin θ)

基本关系

  • sin²θ + cos²θ = 1
  • tan θ = sin θ / cos θ

特殊角的三角函数值

θ 0° 30° 45° 60° 90°
sin θ 0 1/2 √2/2 √3/2 1
cos θ 1 √3/2 √2/2 1/2 0
tan θ 0 √3/3 1 √3 不存在

📚 本章内容

本章综合训练不等式的各类问题：

1. 一元一次不等式

ax + b > 0

2. 一元二次不等式

ax² + bx + c > 0

3. 绝对值不等式

|x| < a ⟺ -a < x < a
|x| > a ⟺ x < -a 或 x > a

4. 分式不等式

f(x)/g(x) > 0 ⟺ f(x)·g(x) > 0

5. 不等式组

取各不等式解集的交集
