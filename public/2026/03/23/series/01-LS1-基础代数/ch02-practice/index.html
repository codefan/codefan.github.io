例题 1：充要条件的判断

题目来源: 课本习题

题意分析:

判断下列各题中 p 是 q 的什么条件：

  1. p: x > 2, q: x > 0
  2. p: x² > 4, q: x > 2
  3. p: 三角形是等边三角形，q: 三角形是等角三角形
  4. p: a 是偶数，q: a² 是偶数

解题思路:

对每个小题：

  1. 判断 p → q 是否为真
  2. 判断 q → p 是否为真
  3. 根据真值表得出结论

参考答案:

  1. p: x > 2, q: x > 0

    • p → q: 若 x > 2，则 x > 0 ✅（真）
    • q → p: 若 x > 0，则 x > 2 ❌（假，如 x=1）
    • 结论: 充分不必要条件

  2. p: x² > 4, q: x > 2

    • p → q: 若 x² > 4，则 x > 2 ❌（假，如 x=-3）
    • q → p: 若 x > 2，则 x² > 4 ✅（真）
    • 结论: 必要不充分条件

  3. p: 等边三角形，q: 等角三角形

    • p → q: 等边 → 等角 ✅（真）
    • q → p: 等角 → 等边 ✅（真）
    • 结论: 充要条件

  4. p: a 是偶数，q: a² 是偶数

    • p → q: 偶数 → 平方是偶数 ✅（真）
    • q → p: 平方是偶数 → 原数是偶数 ✅（真，奇数的平方是奇数）
    • 结论: 充要条件

📊 本章小结

重点回顾

  1. 三种条件的定义：

    • 充分条件：p → q
    • 必要条件：q → p
    • 充要条件：p ↔ q

  2. 判定方法：

    • 定义法：直接判断两个方向
    • 集合法：看集合包含关系
    • 等价法：利用逆否命题

  3. 四种分类：

    • 充要条件（✅✅）
    • 充分不必要（✅❌）
    • 必要不充分（❌✅）
    • 既不充分也不必要（❌❌）

  4. 信竞应用：

    • 二分答案的充要条件
    • 图论判定（欧拉回路、二分图）
    • 贪心算法的正确性证明
    • 搜索剪枝的必要条件

易错点提醒

⚠️ 充分 vs 必要：

  • “如果 p，那么 q” → p 是 q 的充分条件
  • “只有 p，才 q” → p 是 q 的必要条件

⚠️ 集合关系方向：

  • A ⊆ B → “x∈A” 是 “x∈B” 的充分条件
  • 方向容易搞反！

⚠️ 逆否命题等价：

  • p → q 等价于 ¬q → ¬p
  • 直接证明困难时，可以证明逆否命题

⚠️ 必要条件用于剪枝：

  • 不满足必要条件 → 一定无解 → 可以剪枝
  • 满足必要条件 → 不一定有解 → 继续搜索
