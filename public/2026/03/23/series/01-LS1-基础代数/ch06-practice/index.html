例题 1：平移变换

题目来源: 课本习题

题意分析:

将函数 y = x² 的图像向右平移 2 个单位，再向上平移 3 个单位，求变换后的函数解析式。

解题思路:

根据平移规则：

  • 向右平移 2：x → x - 2
  • 向上平移 3：y → y - 3

例题 2：对称变换

题目来源: 课本习题

题意分析:

已知函数 f(x) = 2x + 1，求：

  1. 关于 y 轴对称的函数
  2. 关于 x 轴对称的函数
  3. 关于原点对称的函数

解题思路:

根据对称变换规则：

  • 关于 y 轴对称：f(x) → f(-x)
  • 关于 x 轴对称：f(x) → -f(x)
  • 关于原点对称：f(x) → -f(-x)

参考答案:

  1. 关于 y 轴对称：

    • f(-x) = 2(-x) + 1 = -2x + 1

  2. 关于 x 轴对称：

    • -f(x) = -(2x + 1) = -2x - 1

  3. 关于原点对称：

    • -f(-x) = -(-2x + 1) = 2x - 1

对比系数：

  • 振幅：1 → 2（纵向拉伸 2 倍）
  • 频率：1 → 3（横向压缩为 1/3）

📊 本章小结

重点回顾

  1. 平移变换：左加右减，上加下减
  2. 对称变换：y 轴对称 f(-x)，x 轴对称 -f(x)，原点对称 -f(-x)
  3. 伸缩变换：横向 f(ωx)，纵向 Af(x)
  4. 复合变换：注意顺序，一般形式 y = A·f(ωx+φ)+h

易错点提醒

⚠️ 平移方向：f(x-a) 是向右平移 a，不是向左

⚠️ 变换顺序：先伸缩后平移，还是先平移后伸缩，结果不同

⚠️ 符号处理：关于原点对称是 -f(-x)，不是 -f(x)
