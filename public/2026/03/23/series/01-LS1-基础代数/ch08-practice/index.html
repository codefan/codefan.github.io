[练习] 08-幂函数与对勾函数应用

🎯 例题精讲

例题 1：幂函数的性质

题目来源: 课本习题

题意分析:

比较下列各组数的大小：

  1. 2³ 和 3²
  2. 2^0.5 和 2^0.3
  3. (-2)³ 和 (-2)²

参考答案:

  1. 2³ = 8, 3² = 9，所以 2³ < 3²
  2. 因为 y = 2^x 在 R 上递增，0.5 > 0.3，所以 2^0.5 > 2^0.3
  3. (-2)³ = -8, (-2)² = 4，所以 (-2)³ < (-2)²

📊 本章小结

重点回顾

  1. 幂函数：f(x) = x^a，a 为常数
  2. 对勾函数：f(x) = x + a/x，最小值 2√a
  3. 基本不等式：a + b ≥ 2√(ab)
  4. 快速幂：O(log n) 计算幂

易错点提醒

⚠️ 负指数：x^(-n) = 1/x^n

⚠️ 定义域：对勾函数 x ≠ 0

⚠️ 等号条件：基本不等式等号当且仅当 a = b
