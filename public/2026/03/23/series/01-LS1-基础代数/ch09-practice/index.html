🎯 例题精讲

例题 1：指数运算

题目来源: 课本习题

题意分析:

计算：

  1. 2³ × 2⁴
  2. (3²)³
  3. 5⁰
  4. 2⁻³

参考答案:

  1. 2³ × 2⁴ = 2^(3+4) = 2⁷ = 128
  2. (3²)³ = 3^(2×3) = 3⁶ = 729
  3. 5⁰ = 1
  4. 2⁻³ = 1/2³ = 1/8

📊 本章小结

重点回顾

  1. 指数运算法则：乘法、除法、幂的幂
  2. 指数函数：f(x) = a^x，a > 0 且 a ≠ 1
  3. 快速幂：O(log n) 计算幂
  4. 模意义下的指数：费马小定理、欧拉定理

易错点提醒

⚠️ 负指数：a⁻ⁿ = 1/aⁿ

⚠️ 0 的 0 次幂：无定义

⚠️ 底数范围：指数函数要求 a > 0 且 a ≠ 1
