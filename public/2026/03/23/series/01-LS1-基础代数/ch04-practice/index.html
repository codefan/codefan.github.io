🎯 例题精讲

例题 1：判断函数单调性

题目来源: 课本习题

题意分析:

判断下列函数的单调性：

  1. f(x) = 3x - 2
  2. f(x) = -2x + 1
  3. f(x) = x²
  4. f(x) = 1/x

解题思路:

根据函数类型和图像特征判断。

参考答案:

  1. f(x) = 3x - 2

    • 一次函数，斜率 k = 3 > 0
    • 在 R 上单调递增

  2. f(x) = -2x + 1

    • 一次函数，斜率 k = -2 < 0
    • 在 R 上单调递减

  3. f(x) = x²

    • 二次函数，开口向上
    • 在 (-∞, 0] 上单调递减
    • 在 [0, +∞) 上单调递增

  4. f(x) = 1/x

    • 反比例函数
    • 在 (-∞, 0) 上单调递减
    • 在 (0, +∞) 上单调递减
    • 注意：不能说在定义域上单调（因为不连续）

📐 数学题

习题 1（基础题 ⭐）

判断下列说法是否正确：

  1. 如果 f(x) 在 [a, b] 上单调递增，则 f(a) < f(b)
  2. 如果 f(x) 在 R 上单调递增，g(x) 在 R 上单调递增，则 f(x) + g(x) 在 R 上单调递增
  3. 如果 f(x) 在 R 上单调递增，则 -f(x) 在 R 上单调递减
  4. 单调函数的图像一定是一条直线

📊 本章小结

重点回顾

  1. 单调性定义：x₁ < x₂ → f(x₁) < f(x₂)（递增）
  2. 严格 vs 非严格：严格递增 vs 单调不减
  3. 单调区间：函数可能在某些区间递增，某些区间递减
  4. 判定方法：定义法、图像法、导数法
  5. 信竞应用：二分查找、三分法、单调栈、LIS

易错点提醒

⚠️ 定义域：讨论单调性前，先确定定义域

⚠️ 区间表示：单调区间用区间表示，不是点

⚠️ 不连续函数：f(x) = 1/x 在两个区间分别递减，但不能说在定义域上递减

⚠️ 复合函数：单调性可能改变
