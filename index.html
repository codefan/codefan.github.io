📖 关于这个博客

这个博客是我多年技术生涯的一个新起点。作为一名拥有 20 多年经验的软件工程师，我经历过：

  • 从 C++ 到 Java 的技术栈变迁
  • 从 C/S 架构到 B/S 架构的演进
  • 从单体应用到微服务的转型
  • 从本地部署到云原生的跨越

在这里，我会记录：

  1. 技术实践 - 工作中遇到的真实问题和解决方案
  2. 架构思考 - 对软件设计的理解和心得
  3. 学习笔记 - 持续学习过程中的记录
  4. 生活随笔 - 工作之外的思考和感悟
